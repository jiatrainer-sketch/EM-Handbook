## 🎯 Indication / Role

### ใช้สำหรับ

- **STEMI / NSTEMI** ที่ ongoing ischemia + SBP ≥ 100
- **Acute decompensated heart failure / pulmonary edema**
- **Hypertensive emergency + ACS / CHF** (combo)
- **Refractory angina**

### Mechanism

- **Organic nitrate** → NO donor → cGMP ↑ → smooth muscle relax
- **Venodilator เด่น** (low dose) → ↓ preload → ↓ pulmonary congestion
- **Arterial dilator** (high dose > 100 mcg/min) → ↓ afterload
- **Coronary vasodilator** → ↑ O₂ supply + ↓ demand (dual benefit in ACS)

### ≠ First-line vasodilator ใน hypertensive emergency (general)

- ดีที่สุดเมื่อมี **cardiac context** (ACS, CHF) ร่วม
- ถ้า HT emergency เดียวๆ → Nicardipine ดีกว่า

---

## 💊 การเตรียมยา (Thai Standard)

### Standard: **50 mg + D5W 250 mL = 200 mcg/mL**

```
Nitroglycerin 50 mg (5 amp × 10 mg/10 mL) + D5W 250 mL
= 50,000 mcg / 250 mL = 200 mcg/mL
```

### Dilute: **25 mg + D5W 250 mL = 100 mcg/mL**

ใช้ peripheral IV หรือ start low-dose

```
Nitroglycerin 25 mg + D5W 250 mL = 100 mcg/mL
```

### ⚠️ Non-PVC tubing สำคัญ

- **PVC tubing ดูดซับ NTG** → dose ตกลง 30–80%
- ใช้ **polyethylene หรือ glass** tubing
- ถ้าไม่มีให้ใช้ dose สูงกว่าเพื่อชดเชย + tell nurse

### Solvent: D5W หรือ NSS ได้ (preferred: D5W)

### ⚠️ Protect from light ไม่จำเป็น (ต่างจาก nitroprusside)

## ⚡ Dosing Table (weight-independent — titrate ตามอาการ + BP)

### @ Standard prep **200 mcg/mL**

| Dose (mcg/min) | Rate (mL/hr) | Effect |
|---|---|---|
| **10 (start)** | **3** | Venodilation (↓ preload) |
| 20 | 6 | Venodilation |
| **50** | **15** | Venous + early arterial |
| 100 | 30 | Mixed venous + arterial |
| **150** | **45** | Arterial dilation เด่น |
| **200 (usual max)** | **60** | Maximum |

**Formula:** mL/hr = dose_mcg/min × 60 / concentration_mcg/mL  
- 10 × 60 / 200 = **3 mL/hr**

### @ Dilute prep **100 mcg/mL** → ×2 ทุก rate

| Dose (mcg/min) | Rate (mL/hr) |
|---|---|
| 10 | 6 |
| 50 | 30 |
| 100 | 60 |

### Starting dose + titration

- **Start 10–20 mcg/min** (= 3–6 mL/hr @ 200 mcg/mL)
- **Titrate q 5–10 min** by **5–10 mcg/min**
- Target: pain relief (ACS) / symptom resolution (CHF) / SBP 100–140
- **หยุด titrate** เมื่อ SBP &lt; 90–100 (หรือลด rate)

---

## 🚨 Max Dose: **200 mcg/min** ปกติ (= 60 mL/hr @ 200 mcg/mL)

บางแหล่งใช้ถึง 400 mcg/min ใน refractory pulmonary edema — ระวัง hypotension

### ถ้าถึง max แล้วไม่ดีขึ้น:

**ACS context:**
1. ต้องเช็ค **reperfusion strategy** ก่อน — PCI / fibrinolytic
2. ➕ **Add Morphine** 2–4 mg IV (ระวัง ใน inferior MI — worse outcome per signal from CRUSADE)
3. ➕ **β-blocker** ถ้า tachy + BP ok (ไม่ใน RV infarct)
4. **IABP** ถ้า cardiogenic shock ร่วม

**CHF / pulmonary edema:**
1. ➕ **Furosemide** 40–80 mg IV (double home dose)
2. ➕ **NIV (BiPAP)** — first-line + NTG combo
3. ➕ **Morphine** — **ไม่แนะนำแล้วในปัจจุบัน** (worse outcome)
4. Consider **Nesiritide** หรือ **Nitroprusside** drip ถ้า refractory

**HT emergency + ACS:**
- ถ้า BP ยังสูงทั้งที่ max NTG → switch/add **Nicardipine** หรือ **Labetalol**

### Tolerance (hemodynamic)

- Develops ใน **24–48 hr** continuous infusion
- แก้: **Nitrate-free interval** 8–12 hr/day (ถ้า clinical เอื้อ) หรือ switch drug

---

## 🔌 Access

- **Peripheral IV OK** (not vesicant)
- **Non-PVC tubing เท่านั้น** (polyethylene/polyolefin)
- Line ติด filter ขนาด 0.22 micron ถ้ามี

---

## 📊 การติดตาม (Monitoring)

**Continuous:** BP q 5 min ระหว่าง titrate, q 15 min เมื่อ stable; telemetry

**Hourly:** HR, chest pain (score 0–10), pulmonary exam (rales, SpO₂)

**Watch for:**
- **Hypotension** (#1 — ลด rate หรือหยุด + fluid bolus)
- **Reflex tachycardia** (อาจบ่ง RV infarct — STOP NTG ทันที)
- **Headache** (very common — paracetamol OK, ไม่ต้องหยุดยา)
- **Methemoglobinemia** (rare, high dose > 48 hr)
- **Tolerance** (หลัง 24–48 hr)

---

## 🚫 Contraindications (absolute)

```
🚫 SBP < 90 mmHg
🚫 RV infarct (inferior STEMI with RV involvement)
   - Preload-dependent → NTG = cardiac arrest
   - Check V4R: ST elevation ≥ 1 mm = RV infarct
🚫 PDE5 inhibitor recent use:
   - Sildenafil (Viagra) within 24 hr
   - Tadalafil (Cialis) within 48 hr
   - Vardenafil within 24 hr
   → Profound hypotension + death risk
🚫 Hypertrophic obstructive cardiomyopathy (HOCM)
🚫 Severe aortic stenosis (preload-dependent)
🚫 Increased ICP, pericardial tamponade
```

## ⚠️ ข้อควรระวัง (Critical Pitfalls)

<Callout type="danger" title="Nitroglycerin (NTG) Drip — Pitfalls">

**RV infarct = NTG contraindicated**
→ Inferior MI (II, III, aVF) → check V4R
→ ST↑ V4R ≥ 1 mm = RV infarct
→ Preload-dependent → NTG causes hypotension + arrest
→ ให้ fluid bolus แทน + revasc STAT

**PDE5 inhibitor question**
→ ถามทุกราย chest pain (esp. middle-aged male)
→ Sildenafil/Tadalafil + NTG = ถึงตาย
→ ถ้าใช้ → wait 24-48 hr ก่อน NTG

**PVC tubing ดูดซับยา**
→ Dose จริงที่ถึงคนไข้ลดลง 30-80%
→ ใช้ non-PVC tubing (polyethylene)
→ ถ้าไม่แน่ใจ → ขอ pharmacy

**Hypotension = หยุด/ลด NTG, fluid bolus**
→ SBP < 90 → off
→ Usually recovers ใน 5-10 min (short half-life)
→ ถ้าไม่ขึ้น → ตรวจ RV infarct, ให้ fluid, pressor

**Headache = common ไม่ใช่ side effect ร้ายแรง**
→ 50-80% มี headache
→ Paracetamol 500-1000 mg PO
→ อย่าหยุด NTG เพราะ headache ถ้าอาการหลักยังอยู่

**Tolerance 24-48 hr**
→ Effect ลดลง → ต้อง ↑ dose
→ ถ้า CHF stable → wean off / switch to PO ISDN (nitrate-free interval)
→ ถ้า ACS → tolerance ไม่ค่อยสำคัญใน short-term

**อย่าลืม reperfusion ใน STEMI**
→ NTG ≠ reperfusion — symptomatic เท่านั้น
→ Door-to-balloon < 120 min ยังเป็น target
→ ถ้า PCI ไม่ได้ → fibrinolytic

**Morphine + pulmonary edema ปัจจุบันไม่แนะนำ**
→ Evidence: worse outcome (↑ intubation, ↑ mortality)
→ ใช้ NIV + NTG + lasix แทน
→ ถ้ายังจำเป็น → dose ต่ำ + monitor closely

**Pulmonary edema**
→ NIV (BiPAP IPAP 10-15, EPAP 5-8) + NTG + Lasix = winning combo
→ Intubation สำหรับ failure to respond หรือ altered mental

</Callout>

## 📝 คำสั่งพร้อมใช้ (Copy-Ready Order)

```
INDICATION (check):
[ ] STEMI/NSTEMI with ongoing ischemia (SBP ≥ 100)
[ ] Acute CHF / pulmonary edema
[ ] HT emergency + ACS or CHF component
[ ] Refractory angina

⚠️ CHECK FIRST:
[ ] SBP ≥ 100 mmHg
[ ] No RV infarct (V4R normal)
[ ] No PDE5 inhibitor in past 24-48 hr
[ ] No severe AS / HOCM

NTG DRIP:
- Prep: Nitroglycerin 50 mg + D5W 250 mL = 200 mcg/mL
- Non-PVC tubing (polyethylene)
- Start: 10-20 mcg/min = 3-6 mL/hr
- Titrate q 5-10 min by 5-10 mcg/min
- Target: pain relief / symptom resolution / SBP 100-140
- Max: 200 mcg/min = 60 mL/hr

Monitor:
- BP q 5 min during titration, q 15 min stable
- ECG continuous
- Pain score, pulmonary exam hourly
- SpO₂

Hold/reduce if:
- SBP < 90-100 → hold, fluid bolus prn
- HR > 120 reflex tachy

Paracetamol 500 mg PO prn for headache

Adjunct (per indication):
[ ] Furosemide 40-80 mg IV (CHF)
[ ] NIV / BiPAP (pulmonary edema)
[ ] DAPT + heparin (ACS)
[ ] Reperfusion plan: PCI / fibrinolytic

Wean after 24-48 hr as tolerance develops
Transition to PO ISDN + nitrate-free interval
Admit CCU / monitored bed
```
