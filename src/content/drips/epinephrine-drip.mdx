## 🎯 Indication / Role

### การใช้งานแยกตาม context

| Context | Route | Dose |
|---|---|---|
| **Cardiac arrest** | IV push | 1 mg q 3–5 min |
| **Anaphylaxis (first-line)** | IM | 0.3–0.5 mg IM (อัดต้น) |
| **Anaphylaxis refractory** | IV drip | 2–10 mcg/min |
| **Septic shock** (NE refractory) | IV drip | 0.05–0.5 mcg/kg/min |
| **Cardiogenic shock** (low CO) | IV drip | 0.05–0.5 mcg/kg/min |
| **Post-arrest hypotension** | IV drip | 0.1–0.5 mcg/kg/min |
| **Symptomatic bradycardia** (bridge) | IV drip | 2–10 mcg/min |

### Mechanism

- **α₁**: vasoconstriction → ↑ MAP
- **β₁**: ↑ contractility + HR → ↑ CO
- **β₂**: bronchodilation (ช่วย anaphylaxis)

**กว้างกว่า NE** — ทั้ง vasopressor + inotrope ในตัวเดียว; แต่ arrhythmia risk สูงกว่า

## 💊 การเตรียมยา (Thai Standard)

### Cardiac arrest push (ไม่ใช่ drip)

```
Epinephrine 1 mg (1 amp × 1 mg/1 mL)
IV push ตรงๆ ได้เลย — ไม่ต้อง dilute
Flush ตาม 20 mL NSS, ยก arm 10–20 sec
```

### Anaphylaxis IM (first-line)

```
Epinephrine 1:1,000 (1 mg/mL)
Dose: 0.3–0.5 mg = 0.3–0.5 mL IM
Site: anterolateral thigh (vastus lateralis)
Repeat q 5–15 min prn
```

### Standard drip: **4 mg + D5W 250 mL = 16 mcg/mL**

```
Epinephrine 4 mg (4 amp × 1 mg/1 mL) + D5W 250 mL
= 4,000 mcg / 250 mL = 16 mcg/mL
```

Thai hospital standard — ใช้ได้ทั้ง anaphylaxis + shock drip

### Concentrated: **4 mg + D5W 100 mL = 40 mcg/mL**

```
Epinephrine 4 mg (4 amp) + D5W 100 mL = 40 mcg/mL
```

ICU / fluid restriction — **central line เท่านั้น**

### ⚠️ Note

- ใช้ **D5W** เท่านั้น — NSS ลด potency
- ป้องกันแสง — เปลี่ยน bag q 24 hr
- High-alert drug — double-check label + concentration

## ⚡ Dosing Table (BW 60 kg, Epi 4:250 = 16 mcg/mL)

### Drip — mcg/min (anaphylaxis / bradycardia)

| Dose (mcg/min) | Rate (mL/hr) |
|---|---|
| **2 (start anaphylaxis)** | **7.5** |
| 5 | 19 |
| 10 | 37.5 |
| **20 (max anaphylaxis)** | **75** |

**Formula:** mL/hr = dose_mcg/min × 60 / concentration  
- @ 2 × 60 / 16 = **7.5 mL/hr**

### Drip — mcg/kg/min (shock / post-arrest)

| Dose (mcg/kg/min) | mcg/min | Rate (mL/hr) |
|---|---|---|
| **0.05 (start)** | **3** | **11** |
| 0.1 | 6 | 22.5 |
| 0.2 | 12 | 45 |
| 0.5 | 30 | 112.5 |
| **1.0 (high)** | **60** | **225** |

**Formula:** mL/hr = dose × BW × 60 / concentration  
- @ 0.1 × 60 × 60 / 16 = **22.5 mL/hr**

⚠️ ที่ dose สูง ≥ 0.5 mcg/kg/min → แนะนำใช้ concentrated 40 mcg/mL เพื่อลด volume

### Starting dose แยกตาม indication

- **Anaphylaxis drip**: 2–4 mcg/min (= 7.5–15 mL/hr) titrate MAP ≥ 65
- **Septic shock** (NE refractory): 0.05–0.1 mcg/kg/min (= 11–22.5 mL/hr)
- **Post-arrest**: 0.1–0.5 mcg/kg/min (= 22.5–112.5 mL/hr)
- **Bradycardia bridge**: 2–10 mcg/min (= 7.5–37.5 mL/hr)

---

## 🚨 Max Dose

| Context | Max dose | Rate @ 16 mcg/mL |
|---|---|---|
| Anaphylaxis drip | **20 mcg/min** | **75 mL/hr** |
| Shock (septic/cardiogenic) | **1 mcg/kg/min** = 60 mcg/min | 225 mL/hr (→ ใช้ conc สูงขึ้น) |
| Post-arrest | **0.5 mcg/kg/min** = 30 mcg/min | **112.5 mL/hr** |

**ไม่มี strict ceiling** แต่ dose สูงมาก = arrhythmia + ischemia risk ↑↑

### ถ้าถึง max แล้วยังไม่ตอบสนอง:

**Anaphylaxis:**
1. ตรวจสอบ airway + ให้ adjuncts (ดูหน้า anaphylaxis)
2. ➕ **Vasopressin 40 U IV** single dose (anaphylaxis refractory)
3. ➕ **Methylene blue** 1.5–2 mg/kg IV (last resort)
4. 🔍 R/O biphasic anaphylaxis, underdosed epi IM ก่อน drip

**Septic/Cardiogenic shock:**
1. ➕ **Add NE** ถ้ายังไม่ได้ให้ (epi อาจ 2nd line)
2. ➕ **Vasopressin 0.03 U/min**
3. 💊 **Hydrocortisone 200 mg/day** (refractory septic shock)
4. 🔍 Consider **mechanical support** (IABP, ECMO) ถ้า cardiogenic

---

## 🔌 Access

- **Central line preferred** ทุกกรณี (vasoconstrictor สูง)
- **Peripheral ได้** สำหรับ anaphylaxis ฉุกเฉิน (&lt; 1–2 hr) ขนาด 18G ขึ้นไป
- Watch extravasation q 1 hr
- **Extravasation:** Phentolamine 5–10 mg ใน 10 mL NSS subq รอบ site

---

## 📊 การติดตาม (Monitoring)

**Continuous:** ECG (arrhythmia สำคัญมาก), BP q 5 min (a-line preferred), SpO₂

**q 1 hr:** HR + rhythm, UO (target > 0.5 mL/kg/hr), peripheral perfusion, mental status

**q 2–4 hr:** Lactate, electrolytes (K, Mg), digits

**Watch for:**
- **Tachyarrhythmia** — หลัก dose limiter (VT/VF risk)
- **Digital/mesenteric ischemia** (high dose)
- **Myocardial ischemia** (↑ O₂ demand)
- **Hyperglycemia** (β₂ glycogenolysis)
- **Hypokalemia** (β₂ effect)
- **Extravasation necrosis**

---

## ⚠️ ข้อควรระวัง (Critical Pitfalls)

```
🚩 Anaphylaxis: IM ก่อน drip เสมอ
   - 0.3–0.5 mg IM อัดต้น = first-line
   - Drip สำหรับ refractory เท่านั้น
   - IM เร็วกว่า IV drip ในแง่ clinical onset

🚩 Cardiac arrest: 1 mg q 3–5 min (ไม่ใช่ drip)
   - ให้ทุก loop ที่ไม่ shockable (PEA/asystole)
   - Shockable: หลัง shock ครั้งที่ 3 เป็นต้นไป

🚩 High arrhythmia risk
   - VT, VF, rapid AF
   - แก้ K ≥ 4.0, Mg ≥ 2.0 ก่อน
   - ลด dose ถ้า HR > 120 + ectopy

🚩 ห้ามผสม alkaline
   - Bicarb, aminophylline เสื่อมสลาย epi
   - แยก line

🚩 ระวัง hyperglycemia
   - β₂ → glycogenolysis, inhibit insulin
   - Monitor glucose q 2–4 hr, insulin prn

🚩 Extravasation = ischemia/necrosis
   - กระจายยาเร็วกว่า NE
   - Phentolamine เตรียมไว้เสมอ

🚩 ผู้ป่วย MAO inhibitor
   - ↑ effect 10–20x
   - ลด dose มากๆ หรือหลีกเลี่ยง
```

## 📝 คำสั่งพร้อมใช้ (Copy-Ready Order)

```
INDICATION (check):
[ ] Anaphylaxis refractory to IM epinephrine
[ ] Septic shock (NE already on, add epi for cardiac support)
[ ] Post-arrest hypotension
[ ] Symptomatic bradycardia bridge

EPINEPHRINE DRIP:
- Prep: Epinephrine 4 mg + D5W 250 mL = 16 mcg/mL
- Anaphylaxis start: 2 mcg/min = 7.5 mL/hr → titrate MAP ≥ 65
- Shock start: 0.05 mcg/kg/min = 11 mL/hr @ BW 60 kg
- Post-arrest: 0.1 mcg/kg/min = 22.5 mL/hr @ BW 60 kg

Monitor:
- ECG continuous (arrhythmia main concern)
- BP q 5 min while titrating (a-line preferred)
- UO hourly
- K, Mg, glucose q 2–4 hr
- Peripheral perfusion + extravasation site q 1 hr

Max dose reached → follow max dose protocol above
Admit ICU
```
