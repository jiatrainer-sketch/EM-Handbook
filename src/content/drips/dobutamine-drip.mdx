## 🎯 Indication / Role

### ใช้สำหรับ

- **Cardiogenic shock** — low CO + adequate preload (cold-wet shock)
- **Acute decompensated heart failure** ที่ MAP ≥ 65 แต่ CO ต่ำ
- **Septic shock** + myocardial depression (สังเกต: cold extremities, low ScvO₂)
- **Bridge to definitive therapy** (transplant, LVAD, revascularization)

### ≠ Vasopressor — ไม่ขึ้น BP โดยตรง

- **Pure inotrope + mild chronotrope**
- ↑ contractility + ↓ afterload (β₂ vasodilation)
- ผลรวม: ↑ CO, ↓ PCWP — แต่ MAP อาจ drop ถ้า preload ไม่พอ

### Cold-wet shock checklist ก่อนใช้

```
✓ BP ≥ 80 (MAP ≥ 65) — ถ้าต่ำกว่านี้ ต้องเพิ่ม NE ก่อน
✓ Cold extremities / mottling / ↑ lactate
✓ Echo: low EF / dilated LV
✓ Preload adequate (JVD, PCWP > 15 หรือ IVC plethoric)
✗ ถ้า warm extremities + low BP → NE ก่อน (distributive)
```

## 💊 Preparation (Thai standard)

### Standard: **"Dobu 1:1"** = 250 mg + D5W 250 mL = **1,000 mcg/mL**

```
Dobutamine 250 mg (1 vial) + D5W 250 mL
= 250,000 mcg / 250 mL = 1,000 mcg/mL
```

Thai hospital standard — ใช้กว้างที่สุด

### Concentrated: **"Dobu 2:1"** = 500 mg + D5W 250 mL = **2,000 mcg/mL**

```
Dobutamine 500 mg (2 vials) + D5W 250 mL = 2,000 mcg/mL
```

ใช้เมื่อ dose สูง ≥ 10 mcg/kg/min หรือ fluid restriction

### ⚠️ Note

- Thai amp: **Dobutrex 250 mg/vial** (ต้อง reconstitute ก่อนผสม)
- ใช้ D5W, NSS, RL ได้ทั้งหมด
- **ห้ามผสม bicarb** (alkaline ทำลาย dobutamine)
- Stability: 24 hr ที่อุณหภูมิห้อง

## ⚡ Dosing Table (BW 60 kg, Dobu 1:1 = 1,000 mcg/mL)

| Dose (mcg/kg/min) | mcg/min | Rate (mL/hr) |
|---|---|---|
| **2 (start low)** | **120** | **7** |
| 5 | 300 | 18 |
| **7.5** | **450** | **27** |
| **10** | **600** | **36** |
| 15 | 900 | 54 |
| **20 (max)** | **1,200** | **72** |

**Formula:** mL/hr = (dose × BW × 60) / concentration  
- @ 5 × 60 × 60 / 1,000 = **18 mL/hr**

⚠️ ถ้าใช้ Dobu 2:1 (2,000 mcg/mL): ÷2 ทุก rate → 5 mcg/kg/min = 9 mL/hr

### Starting dose

- **Start 2–5 mcg/kg/min** (= 7–18 mL/hr)
- Titrate ทีละ 2.5–5 mcg/kg/min q 10–15 min
- Target: ↑ UO, ↓ lactate, ↑ warm extremities, ↑ mental status
- **ไม่มี MAP target** (dobu ≠ vasopressor) — ถ้า MAP ลดให้ดู preload / เพิ่ม NE

---

## 🚨 Max Dose: **20 mcg/kg/min** (= 72 mL/hr @ Dobu 1:1, BW 60 kg)

**Dose > 15 mcg/kg/min:** tachycardia + arrhythmia risk ↑↑ — ประโยชน์เพิ่มน้อยมาก

### ถ้าถึง max แล้ว CO ยังไม่ดีขึ้น:

1. ⚠️ **ตรวจ ECG** — tachycardia/VT = ลด dose ทันที
2. ⚠️ **ตรวจ preload** — ถ้า hypovolemic = ให้ fluid ก่อน, ไม่ใช่เพิ่ม dobu
3. 🔄 **Consider Milrinone** (PDE-3 inhibitor)
   - ผล inotrope + vasodilate คล้าย dobu แต่ non-adrenergic
   - ดีกว่าในผู้ป่วยที่ใช้ β-blocker มาก่อน (dobu blunted)
4. 🔍 **Evaluate mechanical support:**
   - IABP (↓ afterload, ↑ coronary perfusion) — ถ้า STEMI cardiogenic shock
   - Impella / VA-ECMO — ถ้า refractory cardiogenic shock
5. ➕ **Add NE** ถ้า MAP &lt; 65 (vasopressor + inotrope combo)
6. 🔍 **R/O reversible cause:** ischemia (urgent revasc?), valvular emergency, toxin

### Switch triggers (ก่อนถึง max)

- **HR > 120** หรือ **new arrhythmia** → ลด dose ทันที
- **MAP &lt; 60** แม้ preload ok → add NE (ไม่ใช่เพิ่ม dobu)
- **ไม่ตอบสนอง 30–60 min** → reassess diagnosis

---

## 🔌 Access

- **Central line preferred** (vasoactive ทุกตัว)
- **Peripheral ได้ชั่วคราว** (&lt; 6 hr, dose &lt; 10 mcg/kg/min, 18G proximal vein)
- Dobu บาง prep มี **sulphite preservative** → ระวัง allergic reaction (rare)

---

## 📊 Monitoring

**Continuous:** ECG (tachyarrhythmia — dose limiter หลัก), BP, SpO₂

**Hourly:** HR + rhythm, UO (target > 0.5 mL/kg/hr), mental status, extremity warmth

**q 2–4 hr:** Lactate (target clearance ≥ 10%/2 hr), echo ถ้าทำได้

**Lab:** Troponin (MI), BNP/NT-proBNP, electrolytes (K, Mg)

---

## ⚠️ Critical Pitfalls

```
🚩 Dobutamine ≠ vasopressor
   - ไม่ขึ้น BP โดยตรง — อาจลด BP (β₂ vasodilation)
   - ถ้า MAP < 65 → ต้องมี NE ก่อน แล้วค่อยเพิ่ม dobu
   - "Dobu + NE" combo = cardiogenic shock standard

🚩 Tachycardia = dose limiter #1
   - HR > 120 → arrhythmia risk, ↑ O₂ demand
   - ลด dose แทนที่จะ push ต่อ
   - แก้ K ≥ 4.0, Mg ≥ 2.0 ก่อน

🚩 β-blocker user → blunted response
   - Dobu ทำงานผ่าน β-receptor
   - ผู้ป่วย chronic carvedilol/metoprolol → ต้องใช้ dose สูงขึ้น
   - หรือ switch to Milrinone (PDE inhibitor, ไม่ผ่าน β)

🚩 ไม่ใช่ inotrope ระยะยาว
   - Long-term dobu → receptor downregulation → tolerance
   - ระยะยาว = bridge เท่านั้น (definitive tx: revasc, transplant, LVAD)

🚩 Ischemia ↑ ใน high dose
   - ↑ HR + ↑ contractility = ↑ myocardial O₂ demand
   - ระวังในผู้ป่วย CAD เด่น / active ACS
   - Monitor troponin

🚩 ห้ามผสม bicarb
   - Alkaline pH ทำลาย dobutamine
   - Use separate line or Y-site ที่ compatible

🚩 ตรวจ echo ก่อนเริ่ม ถ้าทำได้
   - EF ต่ำ จริงไหม? (cold-wet)
   - Tamponade? → dobu ไม่ช่วย, ต้อง pericardiocentesis
   - RV failure? → ใช้ inotrope แบบอื่น (inhaled NO, sildenafil)
```

## 📝 Copy-Ready Order

```
INDICATION (check):
[ ] Cardiogenic shock (cold-wet: cold extremities, ↑ lactate, low EF)
[ ] Acute decompensated HF (MAP ≥ 65 but low CO)
[ ] Septic shock + myocardial depression

DOBUTAMINE DRIP (Thai "Dobu 1:1"):
- Prep: Dobutamine 250 mg + D5W 250 mL = 1,000 mcg/mL
- Start: 2–5 mcg/kg/min = 7–18 mL/hr @ BW 60 kg
- Titrate q 10–15 min by 2.5–5 mcg/kg/min
- Target: ↑ UO, ↓ lactate, ↑ warm extremities
- Max: 20 mcg/kg/min = 72 mL/hr

Monitor:
- ECG continuous (tachyarrhythmia = dose limiter)
- BP q 5–15 min
- UO hourly
- Lactate q 2 hr
- Extremity warmth, mental status q 1 hr
- Troponin, BNP/NT-proBNP baseline

If MAP < 65 → Add/increase NOREPINEPHRINE first
If HR > 120 or arrhythmia → Reduce dobu dose
If max reached → See max dose protocol
Admit ICU / CCU
```
