## 🚩 Red Flags — don't discharge without thorough workup

- **T &gt; 39.5°C** (or &lt; 36°C in sepsis)
- **Hemodynamic instability** — SBP &lt; 90, MAP &lt; 65, lactate &gt; 2
- **Altered mental status** → [AMS pathway](./altered-mental-status)
- **Petechiae** / purpura fulminans (meningococcemia, DIC)
- **Neck stiffness + headache + AMS** → [Meningitis](./meningitis-empiric)
- **Recent surgery / invasive device** → ?deep space infection
- **Immunocompromised** (neutropenia, HIV, chemo, transplant)
- **Recent travel** (tropical) — malaria, dengue, enteric fever, leptospirosis
- **Pregnancy** — chorioamnionitis, pyelonephritis
- **Elderly without localizing signs** — often bacteremia without classical signs

→ **Sepsis bundle if qSOFA ≥ 2 or clinically septic**

## 📊 Sepsis Screen

### qSOFA (bedside, ≥ 2 = high risk)

- **Respiratory rate ≥ 22**
- **Systolic BP ≤ 100**
- **Altered mental status** (GCS &lt; 15)

### SOFA (ICU-level)

See [SOFA](./sofa) content page.

### Sepsis-3 definition

- **Sepsis** = suspected infection + ΔSOFA ≥ 2
- **Septic shock** = sepsis + lactate &gt; 2 + pressor requirement despite fluid

## 🧪 การตรวจเบื้องต้น (Initial Workup)

### Bedside (first 10 min)

- **Full vitals** + mental status
- **Head-to-toe exam** for source:
  - HEENT (pharyngitis, sinusitis, otitis, dental)
  - Neck (rigidity, lymphadenopathy)
  - Chest (crackles, consolidation)
  - Abdomen (tenderness, Murphy, CVA tenderness)
  - Skin (rash, cellulitis, IV lines, surgical sites)
  - Joints (swelling, effusion)
  - Perineum / GU
  - Neuro (focal deficit, meningismus)

### Labs (first 60 min)

- **CBC with diff**
- **BMP, lactate**
- **LFT, lipase, coags**
- **CRP / procalcitonin** (helpful for bacterial)
- **Blood culture × 2** (before antibiotics)
- **UA + urine culture**
- **βhCG** (reproductive-age F)
- **ABG** (severity; acid-base)
- **Troponin** (stress myocardial injury in sepsis)

### Imaging / specific tests (based on suspected source)

- **CXR** (pneumonia, heart failure)
- **CT abd/pelvis** (intra-abdominal source)
- **CT head / LP** (meningitis)
- **Echo** (endocarditis if IVDU, fever of unclear source, positive BCs)
- **Viral PCR** (influenza, COVID, dengue if endemic)
- **Malaria smear** (travel history)
- **Typhoid Widal / stool culture** (GI symptoms + travel)
- **HIV, hepatitis** if risk factors

## 🎯 Source Evaluation Checklist

| Site | Suggestive features |
|---|---|
| **Lung** | Cough, SOB, crackles, infiltrate on CXR |
| **Urinary** | Dysuria, CVA pain, pyuria, elderly with confusion |
| **Intra-abdominal** | Abd pain, peritoneal signs, diarrhea, elevated LFT/lipase |
| **Skin / soft tissue** | Erythema, warmth, ulcer, surgical wound |
| **CNS** | Meningismus, AMS, headache |
| **Bone / joint** | Joint swelling, focal pain (esp prosthesis) |
| **Endovascular** | IVDU, endocarditis signs, prosthetic valve, new murmur |
| **Catheter / line** | Recent line, tunnel tenderness |
| **Pelvic (F)** | PID, endometritis, abortion, chorioamnionitis |
| **Unknown** | Broad workup + empiric broad-spectrum |

## 💊 Initial Management (if sepsis)

### Sepsis "1-hour bundle"

Within **1 hour** of recognition:

1. **Measure lactate**
2. **Blood cultures** × 2 (before antibiotics)
3. **Broad-spectrum antibiotics** — within 1 hour
4. **30 mL/kg IV crystalloid** if hypotensive or lactate ≥ 4
5. **Vasopressor** (NE) if MAP &lt; 65 despite fluid

### Empirical antibiotic by suspected source

| Source | Regimen |
|---|---|
| Unknown / septic shock | Piperacillin-tazobactam 4.5 g IV + vancomycin 20 mg/kg IV |
| CAP (severe) | Ceftriaxone 2 g + azithromycin 500 mg |
| HAP / VAP | Piperacillin-tazobactam + vancomycin (if MRSA risk) |
| Intra-abdominal | Piperacillin-tazobactam OR ceftriaxone + metronidazole |
| Urosepsis | Ceftriaxone 2 g; piperacillin-tazobactam if nosocomial |
| Meningitis | Ceftriaxone 2 g + vanco 20 mg/kg + dex 10 mg; + amp if > 50 yr |
| Skin / soft tissue | Cefazolin 1–2 g; add vanco if MRSA / severe |
| Neutropenic | Piperacillin-tazobactam OR cefepime 2 g ± vanco |

### Supportive

- Paracetamol 1 g PO/IV for T &gt; 38.5 (symptomatic only; does not improve outcomes)
- Avoid NSAIDs in sepsis (renal, bleeding)
- O₂ to keep SpO₂ ≥ 92%
- NPO if surgical source suspected
- DVT ppx, stress ulcer ppx (ICU)
- Glucose control (target &lt; 180)

## ⚠️ ข้อควรระวัง (Common Pitfalls)

```
🚩 Fever ≠ always infection
   - DVT/PE, malignancy, drug fever, autoimmune, heat stroke
   - But always rule out infection first

🚩 Elderly may be hypothermic in sepsis
   - T &lt; 36°C = sepsis criterion too
   - Low WBC (not just high) also concerning

🚩 Don't delay antibiotics for imaging
   - If septic, treat empirically within 1 hour
   - Source workup parallel

🚩 Obtain cultures BEFORE antibiotics
   - But don't delay antibiotics if draws difficult
   - 2 peripheral BCs from different sites

🚩 Negative fever workup in neutropenic = still treat
   - Neutropenic fever = medical emergency
   - Empirical antibiotic regardless

🚩 Petechiae / purpura
   - Meningococcemia, DIC, dengue
   - Treat empirically + blood cultures + fluids

🚩 Recent travel — different differential
   - Malaria (smear q 12 hr × 3)
   - Dengue, typhoid, leptospirosis
   - Lower threshold for admit and empirical treatment

🚩 Immunocompromised — low threshold
   - Signs attenuated
   - Admit + empiric broad-spectrum

🚩 Fever in post-op patient
   - Atelectasis (day 1–2), UTI (day 3), DVT/pneumonia (day 4+), wound (day 5+)
   - Don't dismiss — workup based on timing
```
