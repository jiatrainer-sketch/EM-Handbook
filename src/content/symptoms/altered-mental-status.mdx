## 🚩 Red Flags — act in minutes

| Finding | Immediate action |
|---|---|
| **GCS &lt; 8** | Secure airway (intubate) |
| **Hypoglycemia (&lt; 70)** | **D50W 50 mL IV** — check BEFORE anything else |
| **Focal neurologic deficit** | Stroke code → [Stroke pathway](./stroke-pathway) |
| **Fever + AMS** | Meningitis workup → [Meningitis empiric](./meningitis-empiric) |
| **Pinpoint pupils + resp depression** | Naloxone 0.4–2 mg IV |
| **Hypotension + AMS** | Sepsis / shock workup |
| **Seizure activity** | Benzo + AEIOU workup → [Status epilepticus](./status-epilepticus) |

## 🧠 AEIOU-TIPS Mnemonic (differential)

- **A** — Alcohol, Acid-base
- **E** — Electrolytes (Na, Ca, Mg), Endocrine (hypo/hyper- glycemia, thyroid, adrenal), Encephalopathy (hepatic, uremic)
- **I** — Infection (meningitis, sepsis, UTI in elderly, pneumonia)
- **O** — Overdose / toxins (opioid, benzo, TCA, CO, alcohol)
- **U** — Uremia (BUN), Under-perfusion (shock)
- **T** — Trauma, Temperature (hypo/hyper)
- **I** — Infection (repeated for emphasis), Ischemia (stroke, MI)
- **P** — Psychiatric (last — diagnosis of exclusion), Poisoning, Porphyria
- **S** — Seizure (post-ictal), Stroke, Space-occupying lesion

## 🧪 Initial Workup — "STAT panel"

### First 5 minutes (bedside)

1. **Point-of-care glucose** — FIRST
2. **ABC** + vitals + SpO₂
3. **Core temperature** (rectal if concerned)
4. **Pupils** (pinpoint → opioid; fixed/dilated → anticholinergic, hypoxia)
5. **Breath** (ketones, alcohol, fetor hepaticus)
6. **Skin** (diaphoresis, track marks, rash)
7. **Focal neuro exam:** gaze, facial droop, motor, plantar

### First 15 minutes — labs

- **BMP** (Na, glucose, Cr, BUN) + **Mg, Ca, PO₄**
- **CBC**
- **LFT** (hepatic encephalopathy)
- **ABG** (acid-base, CO retention, ammonia useful if hepatic)
- **Ammonia** (hepatic encephalopathy)
- **TSH** (myxedema coma)
- **Cortisol** (adrenal crisis suspicion)
- **Lactate**
- **Toxicology screen** (urine + blood; acetaminophen, salicylate levels)
- **UA + U-cx** (esp elderly)
- **ECG** (arrhythmia, QT for overdose)

### Imaging

- **CT head STAT** — any AMS of unclear cause
  - Bleed, mass, stroke, hydrocephalus
- **LP** if meningitis concern + CT clear (consider empiric antibiotics BEFORE LP if delay)

### Consider

- **Thiamine 100 mg IV** before glucose if alcoholic / malnourished (Wernicke prevention)
- **EEG** if seizure suspected (NCSE — non-convulsive status)

## 💊 Empirical Reversals ("coma cocktail")

### Give if etiology unclear (safe, fast)

- **Glucose D50W 50 mL IV** — only if POC glucose &lt; 70 or unavailable
- **Thiamine 100 mg IV** — before glucose if possibly alcoholic/malnourished
- **Naloxone 0.4 mg IV** (titrate; start low if opioid-dependent to avoid withdrawal)
- **Flumazenil** — AVOID in unknown overdose (can precipitate seizure in mixed OD, chronic benzo use)

### Empirical antibiotics if meningitis suspected

- **Ceftriaxone 2 g IV + vancomycin 20 mg/kg IV**
- + **Ampicillin 2 g IV** if &gt; 50 yr / immunocompromised (Listeria)
- + **Dexamethasone 10 mg IV** before/with first antibiotic dose

Do not delay antibiotics while awaiting LP.

## 🎯 Cause-Specific Actions

| Etiology clue | Action |
|---|---|
| POC glucose &lt; 70 | D50W 50 mL IV → recheck in 15 min |
| Pinpoint pupils + brady-breath | Naloxone 0.4 mg IV, repeat |
| Fever + headache + neck stiff | Empirical antibiotic + steroid → LP |
| Focal deficit + &lt; 4.5 hr | Stroke code, CT, consider tPA |
| Hx liver disease + ammonia ↑ | Lactulose 30 mL PO/NG, rifaximin |
| Low Na + seizure | 3% NaCl 100 mL → [Hyponatremia](./hyponatremia-symptomatic) |
| High glucose + ketones + Kussmaul | DKA pathway → [DKA](./dka) |
| Hypothermia + bradycardia + goiter hx | Myxedema coma — T4 IV + hydrocortisone |
| Fever + agitation + tachycardia + hx thyroid | Thyroid storm → [Thyroid storm](./thyroid-storm) |

## ⚠️ ข้อควรระวัง (Common Pitfalls)

```
🚩 Check glucose FIRST, always
   - 2-second test saves lives + avoids unnecessary workup
   - Hypoglycemia can present as focal deficit (stroke mimic)

🚩 Don't give glucose before thiamine in alcoholics
   - Can precipitate Wernicke
   - Thiamine 100 mg IV first (or concurrent)

🚩 Flumazenil usually contraindicated
   - Mixed overdose (common) → seizure if TCA, cocaine concurrent
   - Benzo-dependent patient → withdrawal seizure
   - Use only with clear hx iatrogenic benzo (e.g., procedural sedation)

🚩 UTI in elderly — common AMS cause
   - UA reflexively
   - But don't stop workup after finding UA +

🚩 Non-convulsive status (NCSE)
   - AMS without obvious seizure
   - EEG if unexplained persistent altered consciousness
   - Twitching, nystagmus, automatisms may be only signs

🚩 Focal deficit ≠ always stroke
   - Hypoglycemia, hyponatremia, Todd paralysis, complicated migraine
   - Correct metabolic derangement + reassess

🚩 Don't delay antibiotics for LP
   - CT before LP if focal deficit / papilledema / immunocompromised
   - Give empirical antibiotic + steroid while waiting
   - LP can be done up to 2 hr after antibiotic without changing yield much

🚩 Hepatic encephalopathy — ammonia correlates poorly with clinical
   - Treat clinically (lactulose), not chase ammonia number

🚩 Elderly + fall + AMS = subdural until proven otherwise
   - CT head low threshold
   - May be insidious over weeks

🚩 Document GCS components separately
   - Eye / Verbal / Motor
   - Trend over time
```
