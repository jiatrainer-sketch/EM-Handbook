## 🚩 Red Flags (life-threatening rash)

- **Hemodynamic instability** + rash → anaphylaxis / sepsis → IM epi 0.3 mg STAT
- **Purpura / non-blanching petechiae** → meningococcemia, DIC, vasculitis
- **Mucosal involvement** (eyes, mouth, genitals) → SJS / TEN / erythema multiforme major
- **Skin pain / tenderness out of proportion** → necrotizing fasciitis, TEN, SSSS
- **Skin sloughing / Nikolsky sign +** → **SJS / TEN** — STAT consult, ICU/burn unit
- **Facial / tongue swelling + stridor** → angioedema with airway risk
- **Fever + rash + lymphadenopathy + organ dysfunction** → DRESS (eosinophils, hepatitis)
- **Petechiae + fever + ill-appearance** → **meningococcemia** until proven otherwise (ceftriaxone 2g IV + LP)
- **Target lesions with mucosal ulcer** → EM major
- **Full-thickness epidermal loss &gt; 30% BSA** → TEN (mortality 25–50%)

## 🧠 Common Causes

### Drug reactions
- **Morbilliform (maculopapular)** — most common drug rash (penicillin, sulfa, NSAID)
- **Urticaria** (wheals, itchy, transient)
- **SJS / TEN** (target lesions, mucosal, sloughing)
- **DRESS** (Drug Reaction with Eosinophilia + Systemic Sx — 2–8 wk after start)
- **AGEP** (acute generalized exanthematous pustulosis)
- **Fixed drug eruption**

### Infection
- **Viral exanthem** (measles, rubella, roseola, parvovirus B19)
- **Bacterial** (scarlet fever, staph scalded skin, meningococcemia, TSS)
- **Rickettsial** (typhus — Thailand endemic! RMSF)
- **Dengue / chikungunya** (maculopapular, petechiae, tourniquet +)
- **Syphilis** (secondary — palms/soles)
- **HIV seroconversion** (morbilliform + flu-like)

### Allergic
- **Contact dermatitis** (localized, well-demarcated)
- **Anaphylaxis** (urticaria + systemic sx) → [Anaphylaxis](./anaphylaxis)
- **Angioedema** (face, lips, tongue, GI — may be ACE-I related)

### Autoimmune / systemic
- **Vasculitis** (palpable purpura)
- **Lupus** (malar, discoid)
- **Psoriasis / eczema** (chronic)

### Infestation
- **Scabies** (intensely itchy, webspaces, worse at night)
- **Lice**
- **Cellulitis / erysipelas** (red, warm, tender)

## 🧪 Initial Workup

### Must-do
- **Vital signs** (fever, hypotension, hypoxia)
- **Skin exam head-to-toe** (distribution, type, Nikolsky sign, mucous membranes)
- **Mucosal check** — eyes (conjunctivitis), mouth (erosions), genitals
- **Lymph nodes + organomegaly** (DRESS clue)
- **Medication history** (drug + timing — DRESS 2–8 wk, SJS 1–4 wk)
- **Travel + sexual + exposure history**
- **Dengue tourniquet test** in febrile traveler/endemic area

### Labs (based on severity / suspicion)
- **CBC + differential** (eosinophils in DRESS, neutropenia in viral/drug)
- **LFT + renal function** (DRESS, vasculitis)
- **CRP / ESR** (infection, vasculitis)
- **Blood culture** if fever + ill
- **Skin biopsy** (if severe, unclear, SJS/TEN, vasculitis)
- **Skin swab / culture** (SSSS, cellulitis, HSV)
- **Specific serology**: HIV, syphilis RPR, dengue NS1/IgM, lupus (ANA)
- **Tzanck / HSV PCR** if vesicular

## 💉 Stabilization (life-threatening rash)

### Anaphylaxis
- **Epinephrine 0.3–0.5 mg IM (1:1000)** STAT — lateral thigh
- Can repeat q5–15 min
- IV access, IVF bolus, O₂
- Second line: antihistamine, steroid, bronchodilator
- **Observe ≥4–6 hr for biphasic reaction** → [Anaphylaxis](./anaphylaxis)

### Meningococcemia (purpura + fever + ill)
- **Ceftriaxone 2 g IV STAT** (don't wait for LP)
- Contact precaution + post-exposure ppx for close contacts
- LP + blood culture + NP swab

### SJS / TEN
- **Stop offending drug** immediately
- Burn unit / ICU referral (BSA >10% = TEN)
- Aggressive fluid + electrolyte (like burn)
- Wound care, eye care (ophthalmology), nutrition
- **Avoid steroids** (controversial, may ↑ infection)
- IVIG, cyclosporine — specialist decision

### DRESS
- **Stop offending drug**
- **Prednisolone 1 mg/kg/day** (taper over weeks)
- Supportive + treat organ involvement

### Angioedema (with airway threat)
- **Prepare airway** (fiberoptic intubation, cricothyroidotomy kit)
- If ACE-I related: **C1-inh / icatibant** (if available) — steroid/antihistamine limited benefit
- Allergic: epi + antihistamine + steroid

## 🎯 Cause-Specific Treatment

| Cause | Treatment |
|---|---|
| Morbilliform (drug) | Stop drug, antihistamine, symptomatic |
| Urticaria | Cetirizine 10 mg PO; loratadine; severe → prednisolone 30 mg × 5 d |
| Cellulitis | Dicloxacillin / cephalexin PO; vanco if MRSA/severe |
| Scabies | Permethrin 5% topical, treat household |
| Contact dermatitis | Topical steroid, remove trigger |
| Herpes zoster | Acyclovir 800 mg PO 5×/day × 7 d (start &lt;72 hr) |
| Dengue | Supportive, platelet monitor, fluid management |

## ⚠️ Common Pitfalls

```
🚩 Missing meningococcemia
   - Petechiae + fever + any illness → CTX + LP now
   - Don't wait for "classic triad"

🚩 Missing SJS/TEN early
   - Starts flu-like; painful skin = major clue
   - Drug started 1–4 wk ago? Stop + specialist

🚩 Steroids in SJS/TEN
   - Controversial — may worsen infection
   - Burn unit decision, not ED

🚩 Angioedema from ACE-I
   - Antihistamine + steroid don't work well
   - Focus on airway; C1-inh/icatibant if available

🚩 "Allergic" purpura
   - Purpura ≠ hives — think vasculitis, DIC, infection

🚩 Pediatric rash + fever
   - Kawasaki (fever ≥5d + conjunctivitis + rash + lips + lymph node + extremity)
   - MIS-C (post-COVID)
   - Henoch-Schönlein (palpable purpura legs)

🚩 Dengue in endemic area (TH)
   - Tourniquet test, NS1, Hct trend
   - Warning signs: abd pain, vomiting, fluid accumulation
   - Plasma leak around day 4–6 (critical phase)
```
