## 🚩 Red Flags

- **Bilious vomiting** → small bowel obstruction distal to ampulla
- **Bloody / coffee-ground** → UGIB → [UGIB pathway](./ugib)
- **Projectile** (forceful, no nausea) → ↑ ICP (pediatric: pyloric stenosis)
- **+ Altered mental status / severe headache** → CNS cause (hemorrhage, meningitis, ↑ICP)
- **Abdominal distension + no flatus/stool** → obstruction / ileus
- **Severe dehydration** (hypotension, oliguria, altered sensorium)
- **Peritoneal signs** + vomiting → surgical abdomen
- **Fever + vomiting + headache** → meningitis until proven otherwise

---

## 🧠 Common Causes

### GI
- **Gastroenteritis** (viral / bacterial, diarrhea + cramps)
- **Bowel obstruction** (bilious, distension, no BM)
- **Appendicitis** (periumbilical → RLQ pain, anorexia, fever)
- **Pancreatitis** (epigastric radiating to back, lipase ×3)
- **Cholangitis / cholecystitis** (RUQ, Charcot's triad) → [Cholangitis](./acute-cholangitis)
- **Peptic ulcer disease** (epigastric burning)
- **Hepatitis** (RUQ, jaundice, LFT up)

### CNS
- **↑ ICP** (tumor, hemorrhage, hydrocephalus) — projectile, no nausea
- **Migraine** (unilateral throb, photophobia, recurrent)
- **Meningitis / encephalitis** (fever, HA, nuchal rigidity)
- **Vestibular** (vertigo, BPPV, labyrinthitis)

### Metabolic / Endocrine
- **DKA** → [DKA](./dka) (Kussmaul, fruity breath, BG high)
- **Uremia** (CKD stage 5, ammonia breath)
- **Addisonian crisis** (hypotension, hyperpigmentation, ↓Na/↑K)
- **Hypercalcemia** (stones, bones, groans, moans)

### Drug / toxin
- **Chemotherapy**
- **Opioids, erythromycin, metformin, NSAIDs, digoxin (toxicity)**
- **Alcohol intoxication / withdrawal**

### Obstetric
- **Pregnancy** (hyperemesis gravidarum) — **always check βhCG in reproductive-age ♀**
- **Preeclampsia** (BP ↑, proteinuria)

### Cardiac
- **Inferior MI** (vomiting may be dominant symptom esp. ♀/elderly/DM) → check ECG!

## 🧪 การตรวจเบื้องต้น (Initial Workup)

### Must-do
- **Vital signs** (tachycardia, orthostatic BP = volume depletion)
- **Volume status** (skin turgor, mucous membranes, UOP)
- **Abdominal exam** (distension, peritoneal signs, bowel sounds)
- **βhCG** in reproductive-age female
- **Fingerstick glucose** (DKA / hypoglycemia)

### Labs
- **CBC** (infection, anemia from bleeding)
- **BUN / Cr / electrolytes** (hypokalemia, hyponatremia, metabolic alkalosis from vomiting)
- **LFT + lipase**
- **UA** (ketones, infection, pregnancy)
- **Lactate** if severe
- **ECG** if cardiac concern (esp. elderly/DM inferior MI)

### Imaging (based on suspicion)
- **Upright CXR** (free air = perforation)
- **Abdominal X-ray** (obstruction, ileus, gas pattern)
- **CT abd/pelvis with contrast** — if surgical abdomen, obstruction, mass
- **US** (biliary, pregnancy, OB/Gyn cause)
- **CT head** (if AMS, focal signs, HA)

## 💉 Antiemetic Ladder

### 1st line
- **Ondansetron 4–8 mg IV/PO q8h** (5-HT3) — most ED pts, watch QTc
- **Metoclopramide 10 mg IV q6h** (D2 antagonist) — esp if ileus; avoid in bowel obstruction, Parkinson's, seizure

### 2nd line
- **Prochlorperazine 10 mg IV** or **5–10 mg PO** — vestibular, migraine
- **Promethazine 12.5–25 mg IV/IM/PO** — sedating; IV dilute to avoid tissue damage
- **Droperidol 0.625–1.25 mg IV** — refractory; monitor QTc

### Severe / refractory
- **Dexamethasone 4–10 mg IV** (esp. chemo, postop, migraine adjunct)
- **Haloperidol 0.5–2 mg IV** (gastroparesis, cyclic vomiting)

### Pregnancy-safe
- **Pyridoxine (B6) 10–25 mg PO TID ± doxylamine**
- **Ondansetron** (category B, use if severe)
- **Metoclopramide** (category B)

### ⚠️ Pitfalls
- **QTc prolongation** — ondansetron + droperidol + methadone / quinolone stack
- **Dystonic reactions** from metoclopramide / prochlorperazine → IV diphenhydramine 25–50 mg
- **Avoid metoclopramide in bowel obstruction** (↑ perforation)

---

## 💧 Rehydration

### Mild (tolerating PO)
- Oral rehydration solution (ORS), sips q15 min

### Moderate–severe
- **NS or LRS 500–1000 mL bolus** → reassess → maintenance
- **K replacement** if hypokalemic (usually needed if vomiting >24 hr)
- **Correct acid-base** (vomiting → metabolic alkalosis, hypokalemia, hypochloremia)

---

## 🎯 Cause-Specific Actions

| Cause | First-line action |
|---|---|
| Gastroenteritis | ORT, ondansetron PRN, probiotics if >48h |
| Small bowel obstruction | NG decompression, IV fluid, surgery consult |
| Pancreatitis | NPO, aggressive IVF (LR), analgesia |
| DKA | Insulin drip + K repletion + IVF → [DKA](./dka) |
| ↑ ICP | CT head STAT, neurosurgery, HOB 30°, osmotic Rx |
| Pregnancy HG | Pyridoxine + ondansetron + IVF + thiamine before glucose |
| Inferior MI | ECG, STEMI workup → [STEMI](./stemi) |

## ⚠️ ข้อควรระวัง (Common Pitfalls)

<Callout type="danger" title="Nausea / Vomiting — Pitfalls">

**Missing inferior MI presenting as "GI upset"**
→ Elderly, diabetic, female — atypical presentation
→ Always get ECG for epigastric pain + vomiting

**Missing pregnancy**
→ ALWAYS βhCG in reproductive-age female with vomiting
→ Ectopic, molar, HG all possible

**"Gastroenteritis" as default**
→ Rule out surgical abdomen first
→ Appendicitis in elderly often atypical

**Over-aggressive metoclopramide**
→ Causes akathisia, dystonia
→ Avoid in obstruction, Parkinson's

**Dehydration underestimated in elderly**
→ Skin turgor unreliable
→ Use orthostatic VS, UOP

**Hypokalemia from vomiting**
→ Causes arrhythmias, ileus (worsens vomiting)
→ Replace IV if K &lt;3.0 + symptomatic

</Callout>
