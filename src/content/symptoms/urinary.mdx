## 🚩 Red Flags

- **Anuria / severe oliguria** (UOP &lt; 0.5 mL/kg/hr) → AKI → rule out obstruction
- **Fever + flank pain + dysuria** → **pyelonephritis**, risk of urosepsis
- **Hypotension + fever + UTI signs** → **urosepsis / septic shock** → [Septic shock](./septic-shock)
- **Gross hematuria + flank pain** → stone / mass / clot retention
- **Painless gross hematuria** → malignancy until proven otherwise
- **Acute urinary retention + severe pain** → bladder outlet obstruction → catheter STAT
- **Anuria in pregnancy** → obstruction, preeclampsia, emergency
- **Perineal numbness / saddle anesthesia** → cauda equina → MRI STAT

---

## 🧠 Common Causes

### Infection
- **Uncomplicated UTI** (cystitis) — dysuria, frequency, urgency, suprapubic pain
- **Complicated UTI** (♂, pregnant, DM, immunocompromised, catheter, stone)
- **Pyelonephritis** (fever, flank pain, CVAT, N/V)
- **Urosepsis** (hypotension, AMS, lactate up)
- **Prostatitis** (♂, perineal pain, fever)
- **Epididymitis / orchitis** (scrotal pain + swelling)

### Obstruction
- **Ureteric stone** (colicky flank→groin, hematuria)
- **BPH / urethral stricture** (hesitancy, weak stream, retention)
- **Tumor** (bladder, prostate, ureter)
- **Clot retention** (post-op, anticoag, hematuria)

### Other
- **Bladder / prostate cancer** (painless hematuria)
- **Interstitial cystitis** (chronic, sterile urine)
- **Pregnancy** (frequency normal, UTI more common)
- **Vulvovaginitis** (mimics UTI — check pelvic exam in ♀)
- **STI** (chlamydia, gonorrhea — urethritis)

## 🧪 การตรวจเบื้องต้น (Initial Workup)

### Must-do
- **Vital signs** (fever, hypotension, tachycardia = sepsis)
- **Abdominal + CVA tenderness** (pyelonephritis)
- **Bladder scan / palpation** (retention)
- **Genital exam** (♂: prostate, epididymis; ♀: pelvic if vaginitis suspected)
- **βhCG** in reproductive-age female
- **Urinalysis + microscopy + dipstick**

### Labs
- **Urine culture** (any complicated, pyelonephritis, catheter, ♂)
- **CBC** (infection, anemia from hematuria)
- **BUN / Cr** (AKI from obstruction or sepsis)
- **Blood cultures** × 2 if febrile / septic
- **Lactate** if sepsis
- **PSA** (♂ with retention, not acute)

### Imaging
- **Bedside US** (hydronephrosis, retention, stone)
- **CT abdomen/pelvis without contrast** — stone protocol (gold standard for renal colic)
- **CT with contrast** — if abscess, emphysematous pyelo, mass
- **MRI** — if pregnant + colic (avoid contrast)

## 💊 Empirical Therapy

### Uncomplicated UTI (cystitis, non-pregnant ♀)
- **Nitrofurantoin 100 mg PO BID × 5 days** (1st line; avoid CrCl &lt;30)
- **Fosfomycin 3 g PO single dose** (compliance-friendly)
- **Trimethoprim/sulfamethoxazole DS BID × 3 days** (if local resistance &lt;20%)
- **Cephalexin 500 mg PO QID × 7 days** (pregnancy-safe alternative)

### Pregnancy UTI / asymptomatic bacteriuria
- **Cephalexin 500 mg PO QID × 7 days**
- **Nitrofurantoin** (avoid 3rd trimester)
- **Amoxicillin / Augmentin**
- Avoid: TMP-SMX (1st trimester), fluoroquinolones

### Complicated UTI / Pyelonephritis (outpatient, stable)
- **Ciprofloxacin 500 mg PO BID × 7 days**
- **Levofloxacin 750 mg PO × 5 days**
- TMP-SMX if susceptibility known

### Pyelonephritis (inpatient / severe)
- **Ceftriaxone 2 g IV q24h** (1st line)
- **Piperacillin-tazobactam 4.5 g IV q6–8h** (if nosocomial / recent abx)
- **Ertapenem 1 g IV q24h** (ESBL concern)
- **Vancomycin + cefepime ± metronidazole** (urosepsis empirical)

### Urosepsis
- **Broad-spectrum within 1 hr** (piperacillin-tazobactam or meropenem)
- **Source control**: remove/replace catheter, relieve obstruction STAT
- **IVF + vasopressor** → [Septic shock](./septic-shock)

## 🎯 Cause-Specific Actions

| Cause | First action |
|---|---|
| Uncomplicated UTI (♀) | Nitrofurantoin × 5d, no culture needed |
| Pyelonephritis | Ceftriaxone IV, admit if sepsis/pregnant/comorbid |
| Urosepsis | Abx &lt;1 hr + IVF + vasopressor + source control |
| Ureteric stone | IVF, ketorolac 30 mg IV, tamsulosin 0.4 mg, strain urine |
| Acute retention | Foley catheter STAT, measure residual, α-blocker |
| Anuria + hydro | Urology for obstruction, consider nephrostomy |
| Gross hematuria + clot | 3-way catheter + bladder irrigation |

## ⚠️ ข้อควรระวัง (Common Pitfalls)

```
🚩 Treating asymptomatic bacteriuria in non-pregnant adult
   - Only treat if pregnant or pre-urologic procedure
   - Otherwise = drives resistance

🚩 Missing pyelonephritis in elderly
   - May present as AMS, fall, weakness only (no CVAT, no fever)
   - Low threshold for UA + culture

🚩 Fluoroquinolones in pregnancy
   - Contraindicated — use cephalexin instead

🚩 Missing obstruction behind "UTI"
   - If no response in 48–72 hr, check for obstruction/abscess → CT

🚩 Nitrofurantoin in pyelonephritis
   - Inadequate tissue penetration — cystitis only

🚩 Gross hematuria assumed "stone"
   - Painless hematuria = malignancy work-up (urology + cystoscopy)

🚩 Untreated acute retention
   - Permanent bladder damage if >12–24 hr
   - Catheterize STAT, then work up cause
```
