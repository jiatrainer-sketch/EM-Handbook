## 🎯 Definition

**Life-threatening thyrotoxicosis** — clinical diagnosis (ไม่ใช่แค่ lab)

**Mortality:** 10–30% แม้รักษาถูกต้อง — early recognition + treatment สำคัญ

---

## 📊 Burch-Wartofsky Scale (diagnosis)

| Parameter | Criteria | Points |
|---|---|---|
| **Temp (°C)** | 37.2–37.7 | 5 |
| | 37.8–38.3 | 10 |
| | 38.4–38.8 | 15 |
| | 38.9–39.4 | 20 |
| | 39.4–39.9 | 25 |
| | ≥ 40.0 | 30 |
| **CNS** | Mild (agitation) | 10 |
| | Moderate (delirium, psychosis, lethargy) | 20 |
| | Severe (seizure, coma) | 30 |
| **GI** | Moderate (N/V, diarrhea, abd pain) | 10 |
| | Severe (unexplained jaundice) | 20 |
| **CV — HR** | 99–109 | 5 |
| | 110–119 | 10 |
| | 120–129 | 15 |
| | 130–139 | 20 |
| | ≥ 140 | 25 |
| **CV — HF** | Mild (pedal edema) | 5 |
| | Moderate (rales) | 10 |

| **Atrial fib** | Present | 10 |

### Interpretation

| Score | Likelihood |
|---|---|
| **≥ 45** | **Highly suggestive** — treat ทันที |
| 25–44 | Impending storm — treat aggressive |
| &lt; 25 | Unlikely |

⚠️ Scale **เสริม** clinical — ถ้าสงสัยให้รักษาก่อน รอ lab ได้

## 🔴 Common Precipitants (ถามเสมอ!)

- **Infection** (most common — UTI, pneumonia, sepsis)
- **Surgery / trauma** (esp. thyroid surgery)
- **Iodine load** (contrast CT, amiodarone, kelp)
- **DKA**
- **MI / PE / stroke**
- **Stopped antithyroid** (non-compliance, side effect)
- **Parturition / emotional stress**
- **Radioactive iodine therapy** (rare, 1–2 wk after)

---

## 🔴 Immediate Actions (first 30 min)

1. **ABC + IV access** × 2
2. **Continuous ECG** — AF ~10–35%, risk of arrhythmia
3. **STAT labs:**
   - TSH, free T4, free T3
   - CBC, electrolytes, glucose, BUN/Cr, LFT
   - Blood culture × 2 (hidden infection)
   - CK (if rhabdomyolysis suspect)
   - β-hCG (pregnancy)
   - Troponin (high-output failure)
   - ABG
4. **Cooling** — target 37.5°C
   - **Paracetamol 1 g IV/PO q 6 hr**
   - Cooling blanket, tepid sponge, ice packs
   - ⚠️ **ห้าม Aspirin / NSAIDs** — displaces T4 from TBG → worse
5. **Fluid resuscitation** — NSS 1 L rapid then maintain 200–250 mL/hr
6. **Hypoglycemia check + treat** (high metabolic demand)
7. **Thiamine 100 mg IV** ถ้า alcoholic / malnourished
8. **Empiric antibiotic** ถ้าสงสัย infection precipitant
9. **Admit ICU** + endocrine consult

## 💊 Four-Prong Treatment

### Prong 1 — Block NEW hormone SYNTHESIS

**PTU preferred in storm** (blocks peripheral T4→T3 conversion also)

```
Propylthiouracil (PTU):
  Load: 500-1000 mg PO/NG
  Maintenance: 200 mg q 4 hr PO/NG
  
OR

Methimazole (MMI):
  Load: 60-80 mg PO/NG
  Maintenance: 20-30 mg q 6 hr PO/NG
  (Methimazole ออกฤทธิ์นานกว่า แต่ไม่ block T4→T3)
```

⚠️ ไม่มี IV form — ถ้าคนไข้ NG tube ไม่ได้ → rectal administration (off-label)

### Prong 2 — Block hormone RELEASE (iodine)

**ต้องให้ AFTER PTU/MMI อย่างน้อย 1 hr** — ถ้าให้ก่อน = Jod-Basedow phenomenon (worse)

```
Lugol's solution (5% iodine):
  8 drops (~ 250 mg) PO/NG q 6 hr
  
OR

Potassium iodide (SSKI):
  5 drops (~ 250 mg) PO/NG q 6 hr
  
OR (iodine allergy)

Lithium carbonate: 300 mg PO q 6 hr
  (target level 0.6-1 mEq/L)
```

### Prong 3 — Block PERIPHERAL effects (β-block)

```
Propranolol (preferred — non-selective, blocks T4→T3):
  PO: 60-80 mg q 4 hr (up to 120 mg q 6 hr)
  IV: 0.5-1 mg IV q 10 min, titrate (max ~10 mg)
  Then PO when possible
  
Alternative (if propranolol CI):
  Esmolol drip: 250-500 mcg/kg bolus, 
    then 50-100 mcg/kg/min titrate
    (preferred in HF — ultra-short, titratable)
  
  Metoprolol 5 mg IV q 5 min x 3, 
    then 25-50 mg PO q 6 hr
```

⚠️ **Caution in HF:** ใช้ esmolol drip (titratable) + diuretic + ICU monitoring
⚠️ **ห้าม β-block** ใน severe HF with cardiogenic shock

### Prong 4 — Block T4→T3 CONVERSION + treat adrenal insufficiency

```
Hydrocortisone 100 mg IV q 8 hr
  (หรือ Dexamethasone 2 mg IV q 6 hr)
  Adrenal insufficiency common ใน storm
  + ลด T4→T3 conversion
  Taper เมื่อ storm resolve
```

## 💊 Copy Order — Drug Regimen Summary

```
Thyroid storm drug bundle:

1. PTU 1000 mg PO/NG now, then 200 mg q 4 hr
2. Wait 1 hr, then Lugol's 8 drops PO/NG q 6 hr
3. Propranolol 60-80 mg PO q 4 hr 
   (or Esmolol drip if HF)
4. Hydrocortisone 100 mg IV q 8 hr
5. Paracetamol 1 g IV q 6 hr (cooling)
6. Empiric antibiotic if infection suspect
7. NSS 200 mL/hr maintenance
```

## 📊 การติดตาม (Monitoring)

**Q 15–30 min:** HR, BP, temp, SpO₂, mental status

**Q 4–6 hr:** Electrolytes, glucose, BUN/Cr, Ca, Mg

**Q 12–24 hr:** Free T4, free T3 (expect slow decline), LFT

**Daily:** ECG, CXR, LFT (PTU hepatotox rare but serious)

**Watch for:**
- **HF / pulmonary edema** (high output failure)
- **Arrhythmia** (AF common)
- **Hyperthermia** (target 37.5°C)
- **Agranulocytosis** (from PTU/MMI — check CBC if fever)
- **Hepatotoxicity** (PTU → LFT)
- **Hypoglycemia** (high metabolic demand)

---

## ⚠️ ข้อควรระวัง (Critical Pitfalls)

```
🚩 PTU/MMI ก่อน Iodine
   - ให้ Iodine ก่อน PTU → Jod-Basedow (worse storm)
   - ต้องรออย่างน้อย 1 hr หลัง PTU/MMI ก่อน iodine

🚩 ห้าม Aspirin / NSAIDs ลดไข้
   - NSAID displaces T4 from TBG → free T4 ↑ → worse
   - ใช้ Paracetamol + cooling blanket

🚩 PTU > Methimazole ใน storm
   - PTU blocks peripheral T4→T3 (MMI ไม่)
   - PTU ทำงานเร็วกว่า ใน acute setting
   - Methimazole preferred ใน chronic (pregnancy ยกเว้น 1st trimester = PTU)

🚩 β-block ใน HF = careful
   - Esmolol (ultra-short) preferred ถ้า HF
   - Titrate slowly
   - Diuretic + monitoring
   - Propranolol contraindicated ใน severe HF

🚩 Steroid ไม่ใช่ optional
   - Relative adrenal insufficiency common
   - ให้แม้ไม่มีสัญญาณ adrenal crisis
   - ลด T4→T3 conversion ด้วย

🚩 ตรวจหา precipitant
   - Infection #1 — blood cx, UA, CXR, LP ถ้าสงสัย
   - ทำ CT ด้วย contrast = ระวัง iodine load
   - DKA, MI, PE, stroke

🚩 Cooling aggressive
   - Temp > 40°C = ↑↑ mortality
   - Target 37.5°C
   - Ice packs axilla/groin, cooling blanket

🚩 Thromboprophylaxis
   - AF + bed-bound + hypercoagulable
   - LMWH prophylactic dose
   - ถ้า AF → anticoagulation (CHA2DS2-VASc)

🚩 Monitor agranulocytosis
   - PTU/MMI: 0.3% risk
   - Fever + sore throat → stop drug, CBC
   - ถ้า ANC < 1000 → switch drug or supportive

🚩 Pregnancy
   - 1st trimester: PTU (MMI = congenital anomaly)
   - 2nd-3rd trimester: MMI or PTU
   - Storm → treat as non-pregnant + OB consult

🚩 ถ้า refractory → PLEX
   - Plasmapheresis ลบ hormone ออก
   - Last resort ก่อน thyroidectomy
   - Referral ศูนย์ที่ทำได้
```

## 📝 คำสั่งพร้อมใช้ (Copy-Ready Order)

```
THYROID STORM

Burch-Wartofsky score: ___ /60+ (≥45 = likely)
Precipitant suspected: _____________

Immediate:
1. IV x 2, cardiac monitor, ICU admit
2. Labs: TSH, fT4, fT3, CBC, lytes, glucose, BUN/Cr, LFT, 
   troponin, blood cx x 2, β-hCG, ABG
3. 12-lead ECG
4. CXR
5. Endocrine consult

FOUR-PRONG TREATMENT:

[ ] 1. SYNTHESIS BLOCK:
    PTU 1000 mg PO/NG STAT, then 200 mg q 4 hr
    (or Methimazole 80 mg STAT, then 30 mg q 6 hr)

[ ] 2. RELEASE BLOCK (1 HR AFTER PTU/MMI):
    Lugol's solution 8 drops PO/NG q 6 hr
    (or SSKI 5 drops PO/NG q 6 hr)

[ ] 3. β-BLOCK:
    Propranolol 60-80 mg PO q 4 hr
    (or Esmolol drip if HF present)

[ ] 4. STEROID + T4→T3 BLOCK:
    Hydrocortisone 100 mg IV q 8 hr
    (or Dexamethasone 2 mg IV q 6 hr)

Supportive:
[ ] Paracetamol 1 g IV q 6 hr (target T <37.5°C)
[ ] Cooling blanket, ice packs axilla/groin
[ ] NSS 200 mL/hr IV (maintain UO)
[ ] Dextrose if hypoglycemia
[ ] Thiamine 100 mg IV if malnourished
[ ] Empiric antibiotic if infection suspect
[ ] LMWH thromboprophylaxis
[ ] ⚠️ AVOID: Aspirin, NSAID, contrast CT

Monitor:
- HR, BP, Temp, SpO₂ q 15-30 min
- Lytes q 6 hr, fT4/fT3 q 24 hr
- CBC q 48 hr (watch agranulocytosis)
- LFT q 48 hr
- ECG daily

Refractory → PLEX consideration (refer)
Admit ICU + Endocrine
```
