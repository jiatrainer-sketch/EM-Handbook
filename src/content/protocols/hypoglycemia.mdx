## 🎯 Criteria / Diagnosis

### Whipple's triad (classic diagnosis)
1. **Symptoms of hypoglycemia**
2. **Low BS** ตอนมีอาการ
3. **Relief ด้วย glucose**

### Severity by BS

| Level | BS (mg/dL) | Features |
|---|---|---|
| Level 1 (alert) | &lt; 70 | Symptomatic, self-treat ได้ |
| Level 2 (serious) | &lt; 54 | Neuroglycopenic, need help |
| **Level 3 (severe)** | **&lt; 40** มักมี | **Coma, seizure, focal deficit** |

### Symptoms

**Adrenergic (BS ~55–70):**
- Tremor, palpitation, sweating, anxiety, hunger, paresthesia
- ⚠️ อาจ blunted ใน **autonomic neuropathy**, β-blocker user, elderly, hypoglycemia unawareness

**Neuroglycopenic (BS &lt; 55):**
- Confusion, drowsiness, slurred speech
- Seizure, focal deficit (mimic stroke!)
- Coma

### Common causes (find + fix!)

- **Drugs** (ที่พบบ่อยสุด):
  - Insulin (dose, miss meal, renal failure)
  - **Sulfonylurea** (glibenclamide, glipizide) — long-acting, severe, risk recurrence
  - Meglitinide, alcohol, quinine, pentamidine
- **NPO / starvation** + diabetic on meds
- **Renal failure** (insulin + sulfonylurea clearance ↓)
- **Liver failure** (gluconeogenesis ↓)
- **Sepsis**
- **Adrenal insufficiency** (cortisol ↓)
- **Hypothyroidism** (severe)
- **Insulinoma** (rare, fasting hypo without DM)

## ⚡ การรักษาทันที (Immediate Actions)

### Step 1: Check BS + give glucose ทันที

**ถ้าสงสัย hypo — อย่ารอ lab, treat ได้เลย** (low risk / high benefit)

### Step 2A: Conscious + สามารถกลืน → 15-15 rule

```
15 g oral glucose → recheck BS in 15 min → repeat if still < 70
```

**15 g glucose options:**
- น้ำผลไม้ ½ แก้ว (120 mL)
- น้ำตาล 1 ช้อนโต๊ะ ละลายน้ำ
- กลูโคสเม็ด 4 เม็ด
- น้ำผึ้ง 1 ช้อนโต๊ะ
- ลูกอม 3–4 เม็ด

⚠️ **ห้ามใช้ chocolate/ice cream** — ไขมันทำให้ดูดซึมช้า

### Step 2B: Unconscious หรือกลืนไม่ได้ → IV glucose

**D50W (50% Dextrose):**
- **50 mL IV push** (= 25 g glucose)
- ให้ผ่าน large vein + flush NSS ตาม (irritating)
- Repeat ใน 10–20 min ถ้า BS ยัง &lt; 70

**D10W** (ทางเลือกถ้าไม่มี D50):
- 100–250 mL IV bolus (= 10–25 g)
- ใช้ peripheral ได้ปลอดภัยกว่า D50

### Step 2C: No IV access → Glucagon

**Glucagon 1 mg IM/SC**
- Onset 10–15 min
- ⚠️ ไม่ work ใน **starvation, alcoholic, hepatic failure** (glycogen depleted)
- ไม่ work ใน sulfonylurea-induced hypo

### Step 3: หลัง initial correction → **maintenance**

```
D10W drip @ 50–100 mL/hr
Titrate to keep BS > 100
Check BS q15–30 min until stable, then q1h
```

⚠️ **อย่า D/C หลังกลับมา conscious ทันที** — hypo มักกลับมา (รถแถว sulfonylurea)

### Step 4: Sulfonylurea OD — **admit + octreotide**

Sulfonylurea ทำให้ hypo **prolonged + recurrent** (อาจนาน 24–72 hr)

**Octreotide:**
- **50–100 mcg SC q8h × 24–48 hr**
- ช่วย suppress endogenous insulin secretion
- ลด dextrose requirement + prevent recurrence

**Admit ICU/ward monitor × 24 hr (อย่างน้อย)**

### Step 5: Alcoholic / malnourished → **Thiamine ก่อน glucose**

ถ้าสงสัย thiamine deficiency (chronic alcoholic, severe malnutrition):

**Thiamine 100 mg IV ก่อนให้ glucose**
- ป้องกัน Wernicke's encephalopathy (glucose โดยไม่มี thiamine อาจ precipitate)
- ให้ต่อ q 8 hr × 24–48 hr

## 💊 การเตรียมยา (Drug Preparation, BW 60 kg)

### D50W (50% Dextrose)

```
Dose:      50 mL IV push (= 25 g glucose)
Access:    Large vein, follow with NSS flush
Repeat:    10–20 min if BS < 70
```

**⚠️ Warnings:**
- Irritating → use proximal vein
- Avoid peripheral small IV (necrosis risk if extravasate)
- Hypertonic → can cause phlebitis

**🫘 Renal:** ไม่ต้องปรับ  
**🧓 Elderly:** ระวัง overload (fluid), hyperglycemia

---

### D10W drip (maintenance)

```
Rate:      50–100 mL/hr (= 5–10 g glucose/hr)
Titrate:   keep BS 100–180
Monitor:   BS q 15–30 min until stable, then q 1 hr
```

---

### Glucagon

```
Dose:      1 mg IM/SC
Onset:     10–15 min
Duration:  ~60 min
```

**Contraindications / fail:**
- Pheochromocytoma (severe HTN)
- Insulinoma (rebound hypo)
- **Alcoholic / starvation** (glycogen depleted)
- **Sulfonylurea OD** (doesn't address mechanism)

---

### Octreotide (sulfonylurea OD)

```
Dose:      50–100 mcg SC q 8 hr × 24–48 hr
Role:      Suppress insulin secretion
Admit:     ICU/ward × 24 hr minimum
```

**🫘 Renal (CKD):** ไม่ต้องปรับ  
**🧓 Elderly:** ไม่ต้องปรับ

---

### Thiamine (alcoholic / malnourished)

```
Dose:      100 mg IV BEFORE glucose
Repeat:    100 mg IV q 8 hr × 24–48 hr
```

## ⚠️ ข้อควรระวัง (Critical Pitfalls)

```
🚩 Sulfonylurea hypo = prolonged
   - กลับมาได้ 24–72 hr
   - ต้อง admit + octreotide
   - Never D/C จาก ER

🚩 Alcoholic → thiamine FIRST
   - ถ้าให้ glucose ก่อน → Wernicke's
   - ให้ thiamine 100 mg IV first

🚩 Focal neuro deficit = mimic stroke
   - Check BS ก่อน CT stroke เสมอ
   - Hypo แก้ได้ไว, stroke ต้อง tPA
   - "Time is brain" — but first rule out hypo

🚩 Glucagon fail ใน:
   - Alcoholic (glycogen hmd)
   - Sulfonylurea
   - Liver failure
   - Starvation > 24 hr

🚩 Recurrent hypo — find cause
   - Missed meal?
   - Renal failure?
   - New med?
   - Sepsis?
   - Adrenal insufficiency?

🚩 D50 extravasation → necrosis
   - Large proximal vein
   - Flush NSS after

🚩 DM on long-acting insulin
   - Risk hypo 8–12 hr later
   - ต้อง admit observe
```

## 📝 คำสั่งพร้อมใช้ (Copy-Ready Order)

```
1. Bedside BS STAT (DTX/capillary)
2. If BS < 70 with symptoms, OR unconscious:
   a) Thiamine 100 mg IV (if alcoholic/malnourished) — BEFORE glucose
   b) D50W 50 mL IV push via large vein + NSS flush
      (If no IV: Glucagon 1 mg IM/SC)
   c) Recheck BS in 10–15 min → repeat D50 50 mL prn
3. Once BS > 100:
   - D10W drip @ 50–100 mL/hr
   - Monitor BS q 15–30 min x 2 hr, then q 1 hr
4. Labs: CBC, BUN/Cr, electrolyte, LFT, TSH, cortisol
   (venous sample with hypo event → insulin, C-peptide,
    sulfonylurea screen if unclear cause)
5. Identify cause:
   - Med list (insulin, sulfonylurea, missed meal?)
   - NPO status?
   - AKI/CKD? Sepsis? Adrenal?
6. If sulfonylurea OD:
   - Octreotide 50–100 mcg SC q 8 hr × 24–48 hr
   - Admit ICU/ward monitor × 24 hr
7. Alcoholic: admit, thiamine 100 mg IV q 8 hr × 48 hr
8. Adjust DM regimen before discharge:
   - Reduce insulin dose
   - Stop/reduce sulfonylurea
   - Patient education: hypo recognition + 15-15 rule
9. Home glucose meter + glucagon kit prescription
```
