## 🎯 Definition

**Hypertensive emergency** = SBP > 180 **หรือ** DBP > 120 **+ acute end-organ damage**

### End-organ damage ที่ต้องตรวจ

| Organ | Manifestation |
|---|---|
| **Brain** | Stroke (ischemic/hemorrhagic), encephalopathy (headache, confusion, seizure), papilledema |
| **Heart** | ACS, acute HF / pulmonary edema, aortic dissection |
| **Kidney** | AKI (↑ Cr, proteinuria, hematuria) |
| **Eye** | Grade III–IV retinopathy (flame hemorrhage, cotton-wool, papilledema) |
| **Vessel** | Aortic dissection |
| **Pregnancy** | Pre-eclampsia, eclampsia, HELLP |

### ≠ Hypertensive urgency

- **Urgency:** BP > 180/120 **ไม่มี** end-organ damage → ลดช้าๆ 24–48 hr, PO meds
- **Emergency:** มี end-organ damage → IV + ICU ทันที

---

## 🔴 Immediate Actions (first 15 min)

1. **Airway + Breathing + Circulation** — assess mental status, respiratory distress
2. **IV access × 2** (large-bore)
3. **Continuous BP monitoring** — **arterial line preferred** ถ้ามี
4. **Continuous ECG** (telemetry)
5. **12-lead ECG** — screen STEMI, LVH, strain, arrhythmia
6. **CXR** — pulmonary edema, widened mediastinum (dissection)
7. **STAT labs:**
   - CBC, BUN/Cr, electrolytes
   - Troponin (serial)
   - Urinalysis (proteinuria, hematuria, casts)
   - Blood gas (if SOB / altered)
   - β-hCG (female of childbearing age)
8. **Targeted imaging:**
   - CT head non-contrast ถ้า altered / focal deficit / severe headache
   - CT aorta (chest/abd) ถ้า sudden back/chest pain, unequal pulses, pulse deficit
   - Echo ถ้า dissection / HF suspected
9. **Toxicology** — cocaine, amphetamine, withdrawal (alcohol, clonidine)

---

## 🎯 BP Reduction Goals

### General rule

- **First hour:** ↓ MAP **10–20%** (or SBP ↓ 20–25%)
- **Next 2–6 hr:** → 160/100
- **24–48 hr:** → baseline gradually

### ⚠️ แยกตาม etiology

| Context | First-hour target | Rate | Notes |
|---|---|---|---|
| **Ischemic stroke (pre-tPA)** | &lt; 185/110 | 10–20% | ถ้าไม่ได้ tPA: permissive to 220/120 |
| **Ischemic stroke (post-tPA)** | &lt; 180/105 | Maintain × 24 hr | |
| **Hemorrhagic stroke (ICH)** | **SBP &lt; 140** | Rapid | ต่างจาก ischemic |
| **Aortic dissection** | **SBP &lt; 120 + HR &lt; 60 ภายใน 20 min** | Fastest | β-block **ก่อน** vasodilator |
| **ACS** | SBP &lt; 140 (goal) | 10–20% | Avoid NTG ถ้า RV infarct |
| **Pulmonary edema** | ↓ 20–30% SBP | Rapid symptom relief | NTG + Lasix + NIV |
| **Pre-eclampsia** | SBP &lt; 160 + DBP &lt; 110 | MgSO₄ + antihypertensive | |
| **Pheochromocytoma** | Individualized | α-block ก่อน β-block | |
| **AKI / renal** | 10–15% carefully | ไม่เร็วเกิน | ระวัง hypoperfusion |

## 💊 First-line IV Drugs by Etiology

### 1. Stroke (ischemic / hemorrhagic)

- **Nicardipine drip** 5–15 mg/hr (first-line) — see `nicardipine-drip`
- **Labetalol** 10–20 mg IV bolus, repeat q 10 min max 300 mg
- ห้ามใช้: Nitroprusside (↑ ICP), Hydralazine (unpredictable)

### 2. ACS (STEMI / NSTEMI)

- **Nitroglycerin drip** 10–200 mcg/min — see `nitroglycerin-drip`
- **Labetalol** 10–20 mg IV bolus (ถ้าไม่มี HF)
- **Metoprolol** 5 mg IV q 5 min × 3 (ถ้าไม่มี HF)
- ⚠️ ห้าม: short-acting DHP CCB (nifedipine) — reflex tachy → ↑ O₂ demand

### 3. Aortic Dissection

**β-block ก่อน vasodilator เสมอ:**

- **Esmolol** 500 mcg/kg bolus, then 50–300 mcg/kg/min drip → HR < 60
- **Labetalol** 10–20 mg IV q 10 min
- **THEN add Nicardipine** 5–15 mg/hr ถ้ายังไม่ถึง SBP < 120

⚠️ **ห้าม** vasodilator monotherapy — reflex tachy → ↑ shear → extension

### 4. Acute CHF / Pulmonary Edema

- **Nitroglycerin drip** 10–200 mcg/min
- **Furosemide** 40–80 mg IV (or 2× home dose)
- **NIV / BiPAP** first-line
- **Nicardipine** เป็น alternative ถ้า NTG ไม่พอ

### 5. Preeclampsia / Eclampsia

- **MgSO₄** 4–6 g IV bolus over 15–20 min, then **1–2 g/hr** maintenance
- **Hydralazine** 5–10 mg IV q 20 min
- **Labetalol** 10–20 mg IV q 10 min
- **Nicardipine** 5 mg/hr (alternative)
- Delivery = definitive treatment

### 6. Pheochromocytoma Crisis

- **Phentolamine** 5 mg IV q 10–15 min (α-block)
- **THEN** β-block (esmolol) ถ้า tachy
- ⚠️ ห้าม β-block ก่อน α-block — unopposed α → crisis worse

### 7. Sympathetic (cocaine, amphetamine)

- **Benzodiazepine** (diazepam 5–10 mg IV, lorazepam 2–4 mg IV)
- **Nitroglycerin / Nicardipine** drip
- ⚠️ **ห้าม β-block เดียว** — unopposed α → worse ischemia

## 💊 Drug Preparation (BW 60 kg — weight-independent for most)

### Labetalol bolus

```
Amp: 100 mg/20 mL (= 5 mg/mL)
Bolus: 10-20 mg IV push slow (2 min)
Repeat q 10 min, max 300 mg cumulative
Or drip: 100-300 mg + NSS 100 mL, titrate 2 mg/min
```

### Hydralazine bolus

```
Amp: 20 mg/1 mL
Dose: 5-10 mg IV push slow (1-2 min)
Repeat q 20-30 min
Onset 10-20 min, duration 1-4 hr
⚠️ Unpredictable drop + reflex tachy — ไม่ preferred ใน stroke
```

### Esmolol drip (β-selective, ultra-short)

```
Amp: 2.5 g / 10 mL (= 250 mg/mL)
Prep: 2.5 g + D5W 250 mL = 10 mg/mL
Bolus: 500 mcg/kg over 1 min (= 30 mg @ BW 60 kg = 3 mL)
Drip: start 50 mcg/kg/min, titrate q 4 min
  @ BW 60 kg, 50 mcg/kg/min = 3 mg/min = 18 mL/hr
Max: 300 mcg/kg/min
```

### Nitroglycerin drip

See dedicated page — NTG drip card

### Phentolamine (α-block, pheochromocytoma)

```
Amp: 5 mg/vial (reconstitute with 1 mL)
Dose: 5 mg IV push, repeat q 10-15 min
Drip: 1-5 mg/min titrate
```

### MgSO₄ (eclampsia)

```
Loading: 4-6 g (16-24 mL of 25% MgSO₄) IV over 15-20 min
Maintenance: 1-2 g/hr continuous (10% MgSO₄ drip)
Target serum Mg: 4-7 mg/dL
Monitor: DTR, RR ≥ 12, UO ≥ 30 mL/hr
Antidote: Calcium gluconate 10% 10 mL IV slow
```

## 🚫 Drugs to AVOID in HT Emergency

```
🚫 Nifedipine SL / PO fast-acting
   - Unpredictable drop → ischemia (MI, stroke, watershed)
   - ห้ามในทุก emergency
   
🚫 Clonidine IV (no IV form available anyway)

🚫 Nitroprusside ใน stroke (ICH)
   - ↑ ICP จาก cerebral vasodilation
   - Cyanide toxicity ใน long-term use

🚫 Hydralazine ใน aortic dissection, stroke
   - Reflex tachy → worsen dissection
   - Unpredictable drop

🚫 β-blocker alone ใน:
   - Cocaine/amphetamine HT (unopposed α)
   - Pheochromocytoma (unopposed α)

🚫 NTG ใน RV infarct, severe AS, recent PDE5i
```

## ⚠️ Critical Pitfalls

```
🚩 Reduce BP ช้าๆ (general rule)
   - Aggressive drop → watershed stroke, MI, AKI
   - 10-20% MAP drop ใน 1 hr แรกเท่านั้น
   - Exception: aortic dissection (rapid), ICH (rapid)

🚩 Aortic dissection: β-block ก่อน vasodilator
   - Esmolol/Labetalol ก่อน nicardipine
   - ลด HR → ลด shear stress → prevent extension
   - Target: SBP < 120 + HR < 60 ใน 20 min

🚩 ICH vs Ischemic stroke = opposite targets
   - ICH: SBP < 140 (aggressive)
   - Ischemic pre-tPA: < 185/110 (permissive)
   - Ischemic post-tPA: < 180/105 x 24 hr
   - รู้ CT ก่อน BP target

🚩 Pre-eclampsia — MgSO₄ ก่อน antihypertensive
   - MgSO₄ = seizure prophylaxis + mild BP effect
   - Antihypertensive = target SBP 140-150

🚩 ห้ามใช้ Nifedipine SL
   - Sublingual nifedipine = unpredictable crash
   - ยังพบในบาง ER ต่างจังหวัด — ผิดหลัก evidence
   - ใช้ IV แทน (labetalol, nicardipine, hydralazine)

🚩 ตรวจสอบ "reversible" contributors
   - Pain → analgesic
   - Urinary retention → Foley
   - Anxiety / agitation → benzo
   - Alcohol withdrawal → benzo + thiamine
   - Cocaine → benzo + nitrate, ห้าม β-block

🚩 HT urgency ไม่ต้อง IV
   - No end-organ damage → PO captopril/losartan + discharge
   - อย่าเอา IV มาลด BP ใน urgency → hypoperfusion

🚩 Arterial line ถ้าทำได้
   - Cuff BP เวลา titrate drip = ไม่ accurate
   - A-line ดีกว่าใน ICU

🚩 อย่าลืม "Why now?"
   - Missed medication? Intoxication? Pain?
   - Secondary cause? (pheochromocytoma, renal artery stenosis, hyperthyroid, Cushing)
   - Treat precipitant พร้อม BP
```

## 📝 Copy-Ready Order

```
HYPERTENSIVE EMERGENCY

BP @ presentation: ___/___ mmHg
End-organ damage: [ ] stroke [ ] ACS [ ] pulmonary edema
                  [ ] dissection [ ] AKI [ ] eclampsia
                  [ ] retinopathy [ ] encephalopathy

Immediate:
1. IV access x 2, cardiac monitor, arterial line if possible
2. 12-lead ECG
3. CXR
4. CT head if neuro signs; CT aorta if dissection suspect
5. Labs: CBC, BUN/Cr, lytes, troponin (serial), UA, BGA, β-hCG if female

BP TARGET (select):
[ ] General: ↓ 10-20% MAP in 1 hr
[ ] Ischemic stroke (pre-tPA): <185/110
[ ] Ischemic stroke (post-tPA): <180/105
[ ] Hemorrhagic stroke: SBP <140
[ ] Aortic dissection: SBP <120 + HR <60 in 20 min
[ ] Pulmonary edema: 20-30% SBP drop + symptoms resolve
[ ] Pre-eclampsia: SBP <160, DBP <110

DRUG SELECTION (per etiology):
[ ] Nicardipine drip 5-15 mg/hr (stroke, renal, general)
[ ] NTG drip 10-200 mcg/min (ACS, CHF)
[ ] Labetalol 10-20 mg IV bolus q10min (general, stroke)
[ ] Esmolol drip 50-300 mcg/kg/min (dissection — BEFORE vasodilator)
[ ] Hydralazine 5-10 mg IV q20min (preeclampsia)
[ ] MgSO₄ 4-6 g load + 1-2 g/hr (eclampsia)
[ ] Phentolamine 5 mg IV q10min (pheochromocytoma)
[ ] Benzo + NTG (cocaine/amphetamine)

AVOID:
[ ] Nifedipine SL (all cases)
[ ] β-block alone (cocaine, pheochromocytoma)
[ ] NTG (RV infarct, severe AS, recent PDE5i)
[ ] Nitroprusside (ICH — ↑ICP)

Monitor:
- BP q 5-15 min during titration
- ECG continuous
- Neuro checks q 1 hr
- UO hourly
- Troponin serial

Transition to PO meds after stable 24-48 hr
Search for secondary cause if first presentation
Admit ICU / step-down
```
