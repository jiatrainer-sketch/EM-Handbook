## 🎯 Clinical Phenotypes (Warm/Cold × Wet/Dry)

|  | Wet (congestion) | Dry |
|---|---|---|
| **Warm** (perfused) | **Warm-Wet** (most common) — diurese | Warm-Dry — outpatient f/u |
| **Cold** (hypoperfused) | **Cold-Wet** (cardiogenic shock + congestion) — inotrope + diuretic + ICU | Cold-Dry — inotrope, fluid cautious |

### Congestion signs
- Dyspnea, orthopnea, PND
- Crackles, S3, JVD ↑
- Peripheral edema, hepatomegaly
- ↑ BNP / NT-proBNP

### Hypoperfusion signs
- Narrow pulse pressure
- Cool extremities, mottled
- ↓ UOP, ↓ mental status
- ↑ lactate, ↑ Cr

<Callout type="danger" title="Cardiogenic shock (Cold-Wet) = ICU">

SBP &lt; 90 + hypoperfusion + congestion
- IABP or MCS consideration
- Inotrope (dobutamine, milrinone) + vasopressor (NE) as needed
- Address cause (MI → cath lab, tamponade → pericardiocentesis)

</Callout>

## 🧪 Workup

- **ECG** — ischemia, arrhythmia
- **CXR** — cardiomegaly, pulmonary edema, effusion
- **Echo** — ASAP (LVEF, RV, wall motion, valve, pericardium)
- **Labs**: CBC, BMP, Mg, Ca, LFT, TSH, BNP/NT-proBNP, **troponin**, lactate, glucose
- **ABG** if respiratory distress

### BNP cutoffs
- **BNP &lt; 100** pg/mL: HF unlikely
- **BNP &gt; 400–500** pg/mL: HF likely (after adjust age, renal, BMI)
- **NT-proBNP age-adjusted**: &lt; 50 y: 450 / 50–75 y: 900 / &gt; 75 y: 1,800

## 💊 Treatment — LMNOP + Phenotype-guided

### L — Lasix (diuretic)
- **Furosemide 40 mg IV** (naïve) OR **2× home PO dose IV** (chronic user)
- Target UOP 1–2 L / 4–6 hr
- Reassess q 1–2 hr; **double dose if inadequate** (up to 200 mg bolus or infusion)
- **Infusion** if severe: 5–20 mg/hr
- Adjunct: Metolazone 2.5–5 mg PO 30 min before, OR chlorthiazide 500 mg IV, OR spironolactone

### M — Morphine (limited role)
- Small dose (2–4 mg IV) only if severe dyspnea + anxiety
- **Not routine** — associated with ↑ mortality in some studies

### N — Nitrates (for wet, warm, hypertensive)
- **NTG SL 0.4 mg q 5 min**, then **IV drip 10 mcg/min → titrate**
- Target: SBP 140–160 (reduce 10–20%), symptom relief
- Contraindicated: SBP &lt; 90, RV infarct, PDE-5i recent
- Preload + afterload reduction

### O — Oxygen + Position + NIV
- Sit patient upright, legs dangling
- O₂ to SpO₂ &gt; 92%
- **NIV (CPAP 10, or BiPAP)** for severe pulmonary edema — ↓ intubation, ↓ mortality
- Intubate if AMS, fatigue, persistent hypoxemia

### P — Positioning, Pressors, Pumps
- Cardiogenic shock → **NE + dobutamine**
- IABP / Impella / ECMO for refractory

## 🎯 Phenotype-Based Approach

### Warm-Wet (most common)
- **Diuresis** (furosemide IV) + **nitrates** (if HTN)
- **Continue** β-blocker / ACE-I if stable (don't uptitrate)
- Consider hold β-blocker if severe decompensation or cardiogenic

### Cold-Wet (cardiogenic shock + congestion)
- **Inotrope**: Dobutamine 2.5–20 mcg/kg/min OR Milrinone 0.25–0.75 mcg/kg/min (renal adjust)
- **Vasopressor** if SBP low: NE
- **Diuretic** once perfusion restored
- **Address cause**: MI → cath; tamponade → drain; valve → surgery

### Cold-Dry (rare)
- Fluid trial (250 mL NS bolus, reassess)
- Inotrope if still hypoperfused
- Usually end-stage HF

### Warm-Dry
- Oral diuretic adjustment, outpatient management

## 🚨 Precipitants — Always Investigate

**FAILURE mnemonic:**
- **F**orgot medications / non-compliance
- **A**rrhythmia (AFib)
- **I**schemia (MI)
- **L**ifestyle (salt, fluid, alcohol)
- **U**pregulation (thyrotoxicosis, pregnancy, fever)
- **R**heumatic valve / new valvular dysfunction
- **E**mbolism (PE)

Check: ECG, trop, TSH, CBC (anemia), electrolytes, BNP trend

## 💊 Chronic HF Medication Table (HFrEF EF &lt; 40%)

| Class | Example | Notes |
|---|---|---|
| **ACE-I / ARNI** | Sacubitril/valsartan (Entresto) | 1st line GDMT |
| **β-blocker** | Carvedilol, Metoprolol succinate, Bisoprolol | Continue during admission if possible |
| **MRA** | Spironolactone, Eplerenone | K+ monitoring |
| **SGLT-2i** | Dapagliflozin, Empagliflozin | ↓ HF + mortality even non-DM |

These 4 = "**foundational quadruple therapy**" for HFrEF

## ⚠️ Pitfalls

<Callout type="danger" title="AHF Pitfalls">

**Under-diurese** — "watching" HR/BP instead of UOP → prolonged congestion  
→ Reassess q 1–2 hr; double dose if inadequate

**Over-diurese** — overshoot → AKI, low BP, electrolyte disorders  
→ Monitor Cr, K, I/O; slow down if BUN/Cr rising disproportionately

**Nitrate in RV infarct** — ↓ preload → severe hypotension  
→ ECG: inferior MI + V4R → avoid nitrates

**Miss ischemic trigger** — NSTEMI precipitating ADHF  
→ ECG + troponin + consider cath if ischemic

**Restart ACE-I too early in AKI** — worsens renal  
→ Hold ACE-I if Cr ↑ &gt; 30%; restart when stable

**Stop β-blocker unnecessarily** — only if cardiogenic shock or severe bradycardia  
→ Continue if possible; halve dose if mild decompensation

</Callout>

```
1. Upright position, O₂ to SpO₂ > 92%
2. NIV (CPAP 10 cm) if severe pulmonary edema
3. Furosemide 40 mg IV (or 2× home dose); reassess 1 hr
   ± drip 5–20 mg/hr if severe
4. Nitroglycerin SL × 3 → drip 10–100 mcg/min if HTN + wet
5. [Cold-Wet] Dobutamine 2.5 mcg/kg/min → titrate
   [SBP < 90] NE 0.05 mcg/kg/min → titrate
6. Address precipitant:
   - NSTEMI → DAPT + AC + cath consult
   - AFib RVR → rate control (metoprolol / amiodarone)
   - Pneumonia → abx
7. Continue GDMT if stable (β-blocker, ACE-I/ARNI, MRA, SGLT2i)
   Hold/reduce if cardiogenic or AKI
8. Daily weight, strict I/O, telemetry
9. Admit [monitored / ICU if cardiogenic shock or NIV]
```
