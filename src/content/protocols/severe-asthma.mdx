## 🎯 Severity Assessment

### Triage by signs (decide ICU vs ward)

| Parameter | Mild–Mod | **Severe** | **Life-threatening** |
|---|---|---|---|
| **Talk** | Sentences | **Phrases / words** | **Single words / silent** |
| **RR** | &lt; 30 | **≥ 30** | Slow / exhausted / apnea |
| **HR** | &lt; 120 | **≥ 120** | Bradycardia |
| **SpO₂** | ≥ 90% | **&lt; 90%** | **&lt; 92% despite O₂** |
| **PEF** | &gt; 50% best | **≤ 50% best** | **&lt; 33% or unable** |
| **Accessory muscles** | Mild | **Using** | Paradoxical / tired |
| **Mental** | Normal | Anxious | **Altered / drowsy** |
| **Wheeze** | Loud | Loud, biphasic | **Silent chest** |

### Red flags → intubate prep
- **Silent chest** (not moving enough air)
- **Exhaustion** / unable to talk
- **Altered mental status**
- **Cyanosis**, bradycardia
- **SpO₂ &lt; 92% despite high-flow O₂**
- **pH &lt; 7.2** or PaCO₂ rising (normalizing from low is bad — "pseudo-normal")

---

## 🔴 Immediate Actions (first 15 min)

1. **Sit up**, reassure, talk calmly (stress worsens)
2. **O₂ via mask** — target **SpO₂ 93–95%** (ไม่ต้อง 100%)
3. **IV access** + cardiac monitor + SpO₂ continuous
4. **STAT vital signs q 5–15 min**
5. **PEF** ถ้าทำได้ก่อน treatment (baseline)
6. **Start bronchodilator + steroid immediately** (ไม่รอ lab/CXR)
7. **CXR** — R/O pneumothorax, pneumonia (ไม่ต้องรอก่อน rx)
8. **ABG** ถ้า severe / life-threatening (normal CO₂ = concerning!)
9. **CBC, electrolytes, glucose** (hypokalemia จาก β-agonist + steroid)

## 💊 First-Line Treatment (Mild–Moderate → Severe)

### Step 1 — SABA (salbutamol)

```
Salbutamol (Ventolin):
  NB: 2.5-5 mg q 20 min x 3 (first hour), 
      then q 1-4 hr
  MDI + spacer: 4-10 puffs q 20 min x 3, 
      then q 1-4 hr
  (MDI + spacer = equivalent to NB ใน mild-mod; 
   severe → NB)

Continuous NB option (severe):
  Salbutamol 10-15 mg/hr continuous NB
```

### Step 2 — SAMA (ipratropium) — add ใน severe

```
Ipratropium bromide (Atrovent):
  NB 500 mcg q 20 min x 3 (first hour), 
     then q 2-4 hr
  Mix with salbutamol = "Combivent NB"
```

**Rationale:** SABA + SAMA combo = ↓ admission 20% ใน severe (per meta-analysis)

### Step 3 — Systemic Steroid (ทุกคนที่ admit)

```
First choice:
  Prednisolone 40-50 mg PO single dose, 
    then 40 mg/day x 5-7 days
    (PO = IV ใน GI OK คนไข้)

IV options (ถ้ากินไม่ได้):
  Methylprednisolone 40-80 mg IV q 6-12 hr
  Hydrocortisone 100 mg IV q 6 hr
  Dexamethasone 6 mg IV single (alternative, 
    shorter course possible)

⚠️ Onset 4-6 hr — ต้องให้เร็ว
```

### Step 4 — Magnesium Sulfate (severe / refractory)

```
MgSO₄ 2 g IV over 20 min
(1 amp MgSO₄ 50% 4 mL = 2 g diluted ใน NSS 100 mL)

⚠️ Hypotension, flushing, warmth common
⚠️ Monitor DTR, RR (toxicity)
```

**Evidence:** ↓ admission in severe exacerbation; less clear in mild–mod

### Step 5 — Consider (refractory)

```
Terbutaline 0.25 mg SC q 20 min x 3
  (IV not commonly available)

Epinephrine 0.3-0.5 mg IM 
  (if anaphylaxis component, or extreme severe)

Heliox 70:30 (He:O₂)
  (ลด airway resistance — available center only)

IV salbutamol (rarely used)
```

## 💊 Drug Preparation Details (BW 60 kg)

### Salbutamol NB

```
Amp: 2.5 mg/2.5 mL (ready-to-use) หรือ 5 mg/2.5 mL
Dose: 2.5 mg for mild-mod, 5 mg for severe
NB flow O₂ 6-8 L/min
Frequency: q 20 min x 3, then q 1-4 hr

Continuous NB (severe):
  10-15 mg/hr via continuous nebulization
  (30 mL of 0.5% salbutamol in 150 mL saline,
   nebulize continuously)
```

### Ipratropium NB

```
Amp: 500 mcg/2.5 mL
Dose: 500 mcg q 20 min x 3, then q 2-4 hr
Mix with salbutamol in same chamber
```

### Prednisolone PO

```
Tab: 5 mg
Dose: 40-50 mg PO single first day, 
      then 40 mg/day x 5-7 days
⚠️ No taper needed if < 2 wk
Renal: no adjust
Elderly: watch glucose, BP
```

### Methylprednisolone IV

```
Vial: 40 mg, 125 mg
Dose: 40-80 mg IV q 6-12 hr
Dilute in NSS 50 mL, infuse over 15-30 min
Switch to PO prednisolone when able
```

### MgSO₄ 2 g IV

```
Amp: 50% MgSO₄ 10 mL = 5 g
Dose: 2 g = 4 mL of 50% solution
Dilute in NSS 100 mL, infuse over 20 min
⚠️ Monitor DTR, RR, BP
Antidote: Ca gluconate 10 mL IV slow
```

### Epinephrine IM (anaphylaxis component)

```
1:1,000 (1 mg/mL)
Dose: 0.3-0.5 mg = 0.3-0.5 mL IM anterolateral thigh
Repeat q 5-15 min prn
```

### ⚠️ Renal / Elderly
- Salbutamol: no renal adjust; elderly watch tachy/tremor
- Steroid: no renal adjust; elderly watch glucose/BP
- MgSO₄: **reduce ใน renal** (CrCl < 30: halve dose + close monitoring)

## 🫁 When to Intubate

### Indicators (any of):
- **Exhaustion**, decreased effort
- **Altered mental status**, drowsy
- **Silent chest** + rising PaCO₂
- **Cardiovascular instability** (bradycardia, hypotension)
- **pH &lt; 7.2** persistent despite Rx
- **Cardiac arrest imminent**

### Intubation considerations

```
Pre-oxygenate with NIV if able (BiPAP 10/5 brief)
RSI:
  Sedation: KETAMINE 1-2 mg/kg IV (bronchodilator!)
    = 60-120 mg @ BW 60 kg (avoid propofol hypotens)
  Paralytic: Rocuronium 1 mg/kg or Succinylcholine 1.5 mg/kg
ETT size: large (7.5-8.0) for Rx + suction

After intubation:
  - Low rate (8-10/min), low TV (6 mL/kg IBW)
  - High I:E ratio (1:4 or 1:5) — long expiration
  - Allow permissive hypercapnia (PaCO₂ 60-80 OK if pH > 7.2)
  - PEEP 0-5 initially (avoid auto-PEEP)
  - Plateau pressure < 30
  - Continue NB salbutamol in-line
  - Deep sedation + paralytic if severe auto-PEEP
  - ICU admit
```

### Post-intubation: watch for auto-PEEP / dynamic hyperinflation
- Disconnect from vent → chest decompression
- Fluid bolus ถ้า hypotension
- Consider needle decompression ถ้า pneumothorax

## 🚫 Medications to AVOID

```
🚫 Sedatives pre-intubation (benzo, opioid)
   → respiratory depression, worsening
   Only ใน active intubation

🚫 β-blocker (non-selective)
   Even cardioselective ระวัง

🚫 Aspirin / NSAID ถ้ามี AERD history
   (aspirin-exacerbated respiratory disease)

🚫 Over-bagging post-intubation
   → auto-PEEP → hemodynamic collapse

🚫 Routine antibiotic
   Only ถ้าชัดเจน bacterial infection (fever, purulent, CXR)

🚫 Theophylline IV
   Narrow therapeutic, little added benefit, arrhythmia risk

🚫 Chest physiotherapy ใน severe
   Can worsen bronchospasm
```

## ⚠️ ข้อควรระวัง (Critical Pitfalls)

<Callout type="danger" title="Severe Asthma Exacerbation — Pitfalls">

**"Normal" CO₂ = danger sign!**
→ Early asthma: low CO₂ (hyperventilation)
→ Severe: rising to normal (exhaustion, fatigue)
→ Very severe: ↑↑ CO₂, acidemia
→ Don't relax if ABG "looks normal" in severe
→ Correlate with clinical (exhaustion, silent chest)

**Silent chest = severe, not better**
→ Moving so little air that no wheeze
→ Intubate preparation ทันที

**SpO₂ target 93-95%, ไม่ใช่ 100%**
→ Hyperoxia in asthma ไม่ดีต่อ V/Q
→ ลด FiO₂ after stabilize

**Steroid onset 4-6 hr = ให้เร็ว**
→ อย่ารอดูว่า SABA ได้ผลก่อน
→ IV vs PO: equivalent ใน GI OK

**Check + correct K**
→ β-agonist + steroid = hypokalemia
→ Replace to K > 4.0
→ Hypokalemia → arrhythmia risk

**MgSO₄ ใน severe แต่ไม่ใช่ first-line**
→ หลัง SABA/SAMA/steroid
→ Add-on therapy

**CXR ไม่ routine ใน mild-mod**
→ Ordered ถ้า: fever, focal findings,
→ first presentation, failure to respond, pregnancy
→ R/O pneumothorax, pneumonia, pulmonary edema

**Ketamine = preferred RSI agent**
→ Bronchodilator effect (releases catecholamines)
→ Avoid ใน HTN severe (uncommon in asthma)
→ Propofol = hypotension risk

**Ventilator settings ต่างจาก ARDS**
→ Low rate, long expiration, low PEEP
→ Permissive hypercapnia
→ Watch auto-PEEP (trapped gas)

**DNR / advance directive**
→ Severe asthma ที่ historical intubation =
→ high risk future death
→ Discuss early ถ้า recurrent

**Discharge criteria**
→ PEF > 70% personal best
→ Able to talk sentences
→ SpO₂ > 94% room air
→ Able to walk
→ On PO steroid + understanding action plan
→ Inhaler technique verified
→ Follow-up within 2-7 days

**Action plan + trigger assessment**
→ Write asthma action plan
→ Identify + remove trigger (smoke, pet, allergen)
→ Ensure adherence to ICS/LABA
→ Step up controller if repeated exacerbation

</Callout>
