## 🎯 Triage by Duration + Triggers

### AVS (Acute Vestibular Syndrome)
- Continuous vertigo ≥ 24 hr with nystagmus + gait instability + nausea
- **Key DDx**: peripheral (vestibular neuritis, labyrinthitis) **vs central (stroke!)**
- **Use HINTS exam** to differentiate

### Episodic positional
- Brief episodes triggered by head movement (seconds-minutes)
- **BPPV** most common
- **Dix-Hallpike** diagnostic

### Episodic spontaneous
- Episodic, no trigger
- Ménière's (+ hearing loss), vestibular migraine, TIA

---

## 🔑 HINTS Exam (AVS only!)

<Callout type="info" title="HINTS — only for CONTINUOUS vertigo with nystagmus">

Do NOT use HINTS in:
- Episodic vertigo
- No nystagmus present
- Unsteady patient without vertigo

HINTS + audio = **97% sensitive for posterior circulation stroke** (better than early MRI)

</Callout>

### H — Head Impulse Test (h-HIT)
- Patient fixates on your nose
- Rapid head turn 15–20° left/right
- **Normal (peripheral lesion)**: eyes stay on target
- **Abnormal (central/stroke)**: corrective saccade back to target → normal is abnormal!
- Positive = saccade observed = peripheral (vestibular neuritis)

### I — Nystagmus
- **Peripheral (benign)**: unidirectional, horizontal, suppresses with fixation
- **Central (concerning)**: direction-changing with gaze, vertical, pure torsional

### TS — Test of Skew
- Cover-uncover eye test
- **Abnormal (central)**: vertical corrective movement when uncovering = skew deviation

### HINTS rule — "INFARCT" suggests stroke:
- **I**mpulse **N**ormal (no saccade)
- **F**ast-phase **A**lternating (direction-changing nystagmus)
- **R**efixation on **C**over **T**est (skew deviation)

**Any INFARCT finding = central cause → MRI + neurology**

### HINTS plus
- Add: new hearing loss (sudden, unilateral) = **AICA stroke**
- Vertigo + hearing loss ≠ benign labyrinthitis in older; R/O stroke

## 🩺 Dix-Hallpike (BPPV Dx)

### Technique
1. Patient seated, head rotated 45° toward tested side
2. Rapidly lay patient supine with head extending 20° off table
3. Observe eyes for 30 seconds
4. Sit up, observe again

### Positive (posterior canal BPPV)
- **Upbeat + torsional nystagmus** toward ground-side ear
- **Latency** 5–15 sec
- **Duration** &lt; 60 sec
- Fatigues with repetition

### Lateral canal BPPV (10%)
- **Supine roll test** (horizontal nystagmus when rolling head side to side)

---

## 💊 Treatment

### BPPV
- **Epley maneuver** — posterior canal (most common)
  - Confirms BPPV with Dix-Hallpike → proceed immediately with Epley
  - 85–95% success after 1–3 maneuvers
- **Barbeque (Lempert) maneuver** — lateral canal
- Antihistamines (meclizine) only for symptomatic; don't use long-term
- Return precautions; home exercises

### Vestibular neuritis / labyrinthitis
- **Supportive**: rest, antiemetic (ondansetron, metoclopramide)
- **Vestibular suppressant** (meclizine, dimenhydrinate) — 2–3 days only (delays recovery if prolonged)
- **Steroids** (prednisolone 60 mg × 5 d, tapered) — modest benefit in first 24–48 hr
- Vestibular rehabilitation

### Central cause (stroke)
- **Stroke protocol** (see stroke-pathway)
- Posterior circulation: cerebellar, PICA, AICA, basilar
- Thrombolytics within window if ischemic

## 🚨 Red Flags (think central)

<Callout type="danger" title="Red flags — do not miss stroke">

- **New headache** (posterior, neck)
- **Diplopia, dysarthria, dysphagia, dysmetria** (cerebellar)
- **Cranial nerve deficit**
- **Limb weakness / numbness** (even subtle)
- **Gait ataxia** out of proportion to vertigo
- **Age &gt; 60**, HTN, DM, AF, smoking
- **Direction-changing nystagmus**
- **Positive skew test**
- **Negative head impulse** (paradoxical)
- Hearing loss with sudden onset + vertigo

→ **MRI brain (DWI)** + posterior circulation imaging (MRA or CTA)

</Callout>

---

## ⚠️ Pitfalls

<Callout type="warning" title="Vertigo Pitfalls">

**Rely on CT for posterior stroke** — MRI much more sensitive (CT misses ~80%)  
→ CT is poor for posterior fossa; if concern, do MRI

**Misapply HINTS to episodic vertigo** — HINTS only for AVS  
→ Use for continuous vertigo with nystagmus only

**Dismiss elderly vertigo as "BPPV"** — posterior circulation strokes common  
→ Apply HINTS if AVS; MRI for red flags

**Prolonged meclizine use** — delays vestibular compensation  
→ Max 2–3 days; start vestibular therapy early

**Miss cerebellar stroke presenting as isolated vertigo** — rare but critical  
→ Full neuro exam; finger-to-nose, heel-to-shin, gait

</Callout>

## 📝 ER approach

```
Vertigo ED workup

Type of dizziness: [vertigo / presyncope / disequilibrium / other]
Duration: [episodic brief / episodic prolonged / continuous]
Triggers: [positional / none]
Associated: [hearing loss / headache / neuro deficit / CV risk]

Physical:
- Vitals + orthostatic BP
- HEENT (ear, nose, throat)
- Full cranial nerves
- Cerebellar (finger-nose, heel-shin)
- Gait (tandem, single-leg)
- Nystagmus characterization

AVS (continuous) → HINTS:
  □ Head impulse
  □ Nystagmus direction/character
  □ Test of skew

Episodic positional → Dix-Hallpike → Epley if +

If central concern:
  □ MRI brain + MRA/CTA posterior
  □ Neuro consult / stroke activation

If peripheral:
  □ Antiemetic
  □ Brief vestibular suppressant
  □ Steroid for vestibular neuritis
  □ Ear-specific referral if recurrent
  □ Discharge with vestibular rehab
```
