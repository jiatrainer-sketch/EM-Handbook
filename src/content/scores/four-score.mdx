## 🎯 Purpose

ประเมิน **ระดับความรู้สึกตัว** ในผู้ป่วยที่ไม่รู้สึกตัวหรือ intubated — ข้อดีเหนือ GCS คือ **ประเมิน verbal ไม่ได้ก็ยังใช้ได้** และตรวจ brainstem dysfunction ได้

**FOUR = E**ye response + **M**otor response + **B**rainstem reflexes + **R**espiration

---

## 📋 เกณฑ์คะแนน FOUR Score

### E — Eye Response (0–4)

| คะแนน | การตอบสนอง |
|---|---|
| **4** | Eyelids open หรือ opened, tracking หรือ blinking to command |
| **3** | Eyelids open แต่ไม่ tracking |
| **2** | Eyelids closed แต่เปิดได้เมื่อเสียงดัง |
| **1** | Eyelids closed แต่เปิดได้เมื่อกดเจ็บ |
| **0** | Eyelids ปิดสนิท ไม่ตอบสนองต่อ pain |

### M — Motor Response (0–4)

| คะแนน | การตอบสนอง |
|---|---|
| **4** | Thumbs-up, fist, หรือ peace sign ตามคำสั่ง |
| **3** | Localizing to pain (มือขึ้นไปเหนือ clavicle) |
| **2** | Flexion response to pain |
| **1** | Extension response to pain |
| **0** | ไม่มีการตอบสนอง หรือ generalized myoclonus status |

### B — Brainstem Reflexes (0–4)

| คะแนน | การตอบสนอง |
|---|---|
| **4** | Pupil AND corneal reflexes ทั้งสองข้างปกติ |
| **3** | Pupil OR corneal reflex หายไปข้างหนึ่ง |
| **2** | Pupil AND corneal reflexes หายไปทั้งสองข้าง |
| **1** | Pupil หายไปทั้งสองข้าง หรือ corneal หายไปทั้งสองข้าง |
| **0** | Pupil + corneal + cough reflex หายไปทั้งหมด |

### R — Respiration (0–4)

| คะแนน | การตอบสนอง |
|---|---|
| **4** | ไม่ได้ intubated, หายใจสม่ำเสมอ |
| **3** | ไม่ได้ intubated, Cheyne-Stokes breathing |
| **2** | ไม่ได้ intubated, irregular breathing |
| **1** | Intubated, หายใจเองได้ (breathes above ventilator rate) |
| **0** | Intubated, หายใจตาม vent rate เท่านั้น หรือ apnea |

**คะแนนรวม 0–16**

---

## 📊 การแปลผล (Interpretation)

| Score | ความหมาย | แนวทาง |
|---|---|---|
| **16** | ปกติ หรือ nearly normal | ตรวจสาเหตุ |
| **12–15** | Mild impairment | Close monitoring, ICU/HDU |
| **8–11** | Moderate impairment | ICU, หา reversible cause |
| **4–7** | Severe impairment | ICU, aggressive support |
| **1–3** | Very severe | Brain death evaluation, ครอบครัว counseling |
| **0** | Brain death-like | B=0 + R=0 = **brain death-like state** — formal brain death testing |

<Callout type="danger" title="FOUR Score = 0 → ประเมิน Brain Death">

FOUR score 0 (E0 M0 B0 R0) = ไม่มีการตอบสนองใดๆ + apnea  
→ **Brainstem reflexes ทั้งหมดหายไป** = brain death-like  
→ ต้องทำ formal brain death testing ตามโปรโตคอล  
→ แจ้งครอบครัว + ปรึกษา neurology/intensivist

</Callout>

## 🔍 ข้อดีของ FOUR Score เหนือ GCS

| ลักษณะ | GCS | FOUR Score |
|---|---|---|
| **Verbal component** | ต้องพูดได้ | ❌ ไม่ต้องมี — ใช้ได้ใน intubated |
| **Brainstem assessment** | ไม่มี | ✅ ตรวจ pupil + corneal + cough |
| **Locked-in detection** | ❌ อาจพลาด | ✅ E4 M0 = ตรวจพบได้ |
| **Brain death prediction** | ไม่ดี | ✅ Score 0 = brain death-like |
| **Respiratory pattern** | ไม่มี | ✅ ประเมิน respiratory drive |
| **Intubated patients** | Verbal = 1 (V1T) เสมอ | ✅ ประเมินได้ครบ |
| **Ceiling effect** | สูง | ต่ำกว่า |

## 🔍 Locked-In Syndrome — วิธีตรวจ

**ผู้ป่วย Locked-In:** conscious แต่ quadriplegic + anarthric  
→ FOUR: **E4 M0 B4 R1** (หรือ R4 ถ้าไม่ได้ intubated)  
→ GCS: 3 (E1V1M1) — ดูเหมือน coma แต่ **ยังมีสติ!**

```
ตรวจ Locked-In:
  → "ขอให้กระพริบตาหนึ่งครั้งถ้าได้ยิน"
  → "ขอให้กลอกตาขึ้นถ้าได้ยิน"
  → E4 (tracking/blinking) + no motor → สงสัย locked-in
  → MRI brainstem ทันที (pontine lesion)
```

## 💊 การจัดการตาม FOUR Score

### FOUR 12–16 (Mild impairment)
- **Airway:** ประเมิน gag reflex — ถ้าไม่ดี → intubate
- **Monitor:** neuro checks q 1–2 hr
- **Labs:** glucose, electrolytes, ammonia, drug screen
- **Imaging:** CT brain ± MRI ตามสาเหตุ

### FOUR 8–11 (Moderate)
- **ICU admission**
- **Airway protection:** intubate ถ้า GCS ≤ 8 หรือ FOUR ≤ 8 (โดยทั่วไป)
- **Reversible causes:** AEIOU TIPS (alcohol, epilepsy, insulin, opiates, uremia, trauma, infection, psychiatric, stroke/structural)
- **ICP management** ถ้า head CT แสดง mass effect

### FOUR 4–7 (Severe)
- **ICU, intubation**
- **Head of bed 30°** ลด ICP
- **Glucose control:** 140–180 mg/dL
- **Temperature:** normothermia (ป้องกัน secondary brain injury)
- Neurology/neurosurgery consult

### FOUR 0–3 (Brain death-like)

| ยา/การรักษา | ขนาด | Route | Duration | เหตุผล |
|---|---|---|---|---|
| **Mannitol** | 1 g/kg | IV bolus | q 4–6 hr PRN | Reduce ICP (osmotic) |
| **Hypertonic saline (3%)** | 250 mL | IV | over 30 min PRN | Alternative to mannitol |
| **Levetiracetam** | 500–1500 mg | IV | bid | Seizure prophylaxis/treatment |
| **Thiamine** | 100 mg | IV | ก่อน dextrose | Wernicke prevention |
| **Naloxone** | 0.4–2 mg | IV | q 2 min PRN | Suspected opioid |
| **Flumazenil** | 0.2 mg | IV | q 1 min up to 1 mg | Suspected benzo (ระวัง seizure) |

<Callout type="warning" title="Reversible Causes of Coma — ต้องรีบแก้ไข">

**AEIOU TIPS — ตรวจทุกราย:**  
A = Alcohol / Acidosis  
E = Epilepsy / Electrolytes  
I = Insulin (hypo/hyperglycemia)  
O = Opiates / Overdose  
U = Uremia  
T = Trauma  
I = Infection (meningitis/encephalitis)  
P = Psychiatric / Psychogenic  
S = Stroke / Structural  

→ **Thiamine 100 mg IV ก่อนเสมอ** ถ้าสงสัย alcohol/malnourished  
→ Dextrose 50% 50 mL IV ถ้า glucose &lt; 60 mg/dL หรือ unknown

</Callout>

## 🗺 อัลกอริทึม (Algorithm)

```
ผู้ป่วยไม่รู้สึกตัว → FOUR Score ทันที

FOUR 16: อาการเล็กน้อย → ตรวจสาเหตุ
FOUR 12–15: HDU/ICU → monitor + reversible cause
FOUR 8–11: ICU → intubate + CT brain + treat cause
FOUR 4–7: ICU → aggressive support, ICP management
FOUR 1–3: ICU → consider brain death protocol
FOUR 0:
  → Confirm: ไม่มีสาเหตุ reversible?
  → ไม่มียา sedation/paralytic?
  → Core temp ≥ 36°C?
  → → Formal brain death testing
  → → Family meeting + goals of care
```

## ⚠️ ข้อควรระวัง (Critical Pitfalls)

<Callout type="danger" title="FOUR Score — Pitfalls">

**ประเมินขณะ paralytic/heavy sedation**  
→ ต้องหยุดยา + รอผล washout ก่อน FOUR score ใช้ได้  
→ Succinylcholine: รอ 10–15 min; vecuronium: รอ 30–60 min

**พลาด Locked-In Syndrome**  
→ ตรวจ vertical eye movement ทุกราย (FOUR E4 = locked-in possible)  
→ GCS 3 ≠ unconscious เสมอ

**FOUR 0 ≠ Brain death ทันที**  
→ ต้องแยก reversible causes ก่อน (drug, metabolic, hypothermia)  
→ Formal brain death protocol ต้องทำตาม hospital policy

**ลืม glucose ในผู้ป่วย altered consciousness**  
→ Hypoglycemia เป็นสาเหตุ reversible ที่รักษาได้ทันที  
→ ตรวจ bedside glucose ก่อนอย่างอื่น

</Callout>
