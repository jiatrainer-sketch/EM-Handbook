## 🎯 จุดประสงค์ (Purpose)

ประเมินความเป็นไปได้ของ **necrotizing fasciitis (NF)** จากผลแล็บ 6 ตัวแปร — ใช้เป็น **rule-in** ไม่ใช่ rule-out

**⚠️ สำคัญ:** LRINEC ต่ำ **ไม่** ตัด NF ออก — clinical suspicion สูงต้อง consult surgery เสมอ

---

## 📋 เกณฑ์คะแนน (Scoring Criteria)

| พารามิเตอร์ | เกณฑ์ | คะแนน |
|---|---|---|
| **CRP (mg/L)** | &lt; 150 | 0 |
| | ≥ 150 | **4** |
| **WBC (×10³/µL)** | &lt; 15 | 0 |
| | 15–25 | 1 |
| | &gt; 25 | 2 |
| **Hemoglobin (g/dL)** | &gt; 13.5 | 0 |
| | 11–13.5 | 1 |
| | &lt; 11 | 2 |
| **Sodium (mEq/L)** | ≥ 135 | 0 |
| | &lt; 135 | 2 |
| **Creatinine (mg/dL)** | ≤ 1.6 | 0 |
| | &gt; 1.6 | 2 |
| **Glucose (mg/dL)** | ≤ 180 | 0 |
| | &gt; 180 | 1 |

**คะแนนสูงสุด: 13**

---

## 📊 การแปลผล (Interpretation)

| คะแนน | ความเสี่ยง | PPV | การจัดการ |
|---|---|---|---|
| **&lt; 6** | ต่ำ (Low) | ~50% | Monitor close, repeat labs q 6–12 hr ถ้า worsens |
| **6–7** | ปานกลาง (Intermediate) | 50–75% | **Surgical consult urgent** + CT หรือ MRI |
| **≥ 8** | สูง (High) | &gt; 75% | **Emergency surgical exploration STAT** |

<Callout type="danger" title="LRINEC ≥ 6 + สงสัย NF = ศัลยกรรมด่วน">

LRINEC ≥ 6 ร่วมกับ clinical suspicion (ไข้, pain disproportionate, skin changes, crepitus) → **consult surgery STAT**

→ **อย่ารอผล imaging ถ้า hemodynamically unstable**  
→ "The only cure for NF is surgery" — CT/MRI ช่วยยืนยัน แต่ไม่ควร delay OR

</Callout>

---

## 🔑 Clinical Features ที่บ่งบอก NF (นอกเหนือจาก LRINEC)

**"Red flags" ที่ต้องสงสัย NF ทันที:**
- Pain **disproportionate** to appearance
- Rapid progression (&lt; 24 hr)
- **Crepitus** (เสียงดังเวลาคลำ) — เกิดจาก gas ใน fascial plane
- Skin discoloration → gray/black → necrosis
- Bullae formation (ตุ่มน้ำ)
- **Systemic sepsis** (fever, tachycardia, hypotension)
- Numbness/anesthesia ของผิวหนังบริเวณนั้น

**NF ประเภท:**
- **Type I** — polymicrobial (Gram+/−/anaerobes): เบาหวาน, ผู้สูงอายุ, immunocompromised
- **Type II** — Group A Streptococcus (GAS): คนอายุน้อย, ดุร้ายกว่า

## 💊 การรักษา (Treatment)

### การรักษาทางศัลยกรรม (หลักสำคัญ)

**Surgical debridement:**
- **Wide excision** ทันที — ตัดเนื้อตายออกทั้งหมดจนถึงขอบที่มีเลือดสด
- **Second-look surgery** ใน 24–48 hr เสมอ (NF progress ต่อเนื่อง)
- Vacuum-assisted closure (VAC) / wound management หลัง debridement
- Amputation อาจจำเป็นในรายรุนแรง

### ยาปฏิชีวนะ (Antibiotics) — ให้ก่อน OR ทันที

**Type I (polymicrobial) — broad spectrum:**

| ยา | ขนาด | ระยะเวลา |
|---|---|---|
| **Piperacillin-tazobactam** | 4.5 g IV q 6 hr | จนกว่า clinical improvement + defervescence |
| + **Clindamycin** | 600–900 mg IV q 8 hr | ร่วมกัน — ลด toxin production |
| + **Vancomycin** | 25–30 mg/kg/day (div q 8–12 hr) | ถ้าสงสัย MRSA |

**หรือ:**
| ยา | ขนาด | หมายเหตุ |
|---|---|---|
| **Meropenem/Imipenem** | 1 g IV q 8 hr | ถ้าสงสัย ESBL หรือ GN resistant |
| + **Clindamycin** | 600 mg IV q 8 hr | เสมอ — anti-toxin |
| + **Vancomycin** | ตามน้ำหนัก/renal function | ถ้าสงสัย MRSA |

**Type II (GAS) — targeted:**

| ยา | ขนาด | ระยะเวลา |
|---|---|---|
| **Penicillin G** | 4 MU IV q 4–6 hr | ≥ 14 วัน (หรือจนสถานการณ์ดีขึ้น) |
| + **Clindamycin** | 600–900 mg IV q 8 hr | **บังคับ** — Eagle effect (GAS ดื้อ penicillin เมื่อ inoculum สูง; clindamycin ยับยั้ง protein/toxin synthesis) |

<Callout type="warning" title="Clindamycin สำคัญมากใน NF">

Clindamycin ลด **toxin production** ทั้ง GAS (streptococcal pyrogenic exotoxin) และ anaerobes  
→ ให้ร่วมกับ penicillin/pip-tazo เสมอ ไม่ว่า type ใด  
→ ขนาด: 600–900 mg IV q 8 hr; ระยะ: 10–14 วัน (หรือจนสรุปสาเหตุ)

</Callout>

### การรักษาประคับประคอง (Supportive)

**Resuscitation:**
- IV fluid aggressive (LR preferred) — target MAP ≥ 65, UOP ≥ 0.5 mL/kg/hr
- Vasopressor ถ้า refractory (Norepinephrine first-line)
- Intubation ถ้า respiratory failure หรือ GO to OR

**Supportive:**
- Thiamine, folate, MVI (ถ้า alcoholism/malnutrition)
- Tight glycemic control (ถ้า DM): target glucose 140–180 mg/dL
- อาจพิจารณา **IVIG** ใน severe GAS NF (ลด toxin) — dose: 1 g/kg day 1, 0.5 g/kg days 2–3
- **Hyperbaric oxygen (HBO)** — data ไม่ชัดเจน; ไม่ delay surgery เพื่อรอ HBO

## 🗺 อัลกอริทึม (Algorithm)

```
Soft tissue infection + สงสัย NF

↓ Clinical assessment (pain, spread, systemic signs)
↓ Labs: CRP, WBC, Hb, Na, Cr, Glucose → คำนวณ LRINEC

LRINEC < 6 + ไม่มี clinical red flags:
  → Monitor, repeat labs, antibiotic for cellulitis
  → Reassess q 6 hr

LRINEC 6–7 หรือ clinical concern:
  → Surgical consult urgent
  → CT soft tissue (look for gas, fascial thickening)
  → Start broad-spectrum abx ทันที

LRINEC ≥ 8 หรือ clinical red flags อย่างน้อย 1:
  → Surgical consult STAT — อย่า delay OR
  → Broad-spectrum abx IV ทันที
  → ICU admission (sepsis/septic shock likely)
  → Second-look 24–48 hr หลัง debridement
```

## ⚠️ ข้อควรระวัง (Critical Pitfalls)

<Callout type="danger" title="LRINEC Pitfalls">

**LRINEC ต่ำ = ตัด NF ออก** — ผิดอย่างร้ายแรง  
→ LRINEC &lt; 6 มี NPV ~96% เท่านั้น (4% false negative) → ถ้า clinical สงสัย = consult surgery

**รอ imaging ก่อน surgery ในรายไม่ stable** — ห้าม  
→ CT ใช้ได้ถ้า stable; unstable → OR ทันที

**ให้ abx เฉพาะ gram-positive** — ไม่พอ  
→ Type I คือ polymicrobial → broad coverage เสมอ

**ลืม clindamycin** — ขาด anti-toxin mechanism  
→ Clindamycin เป็น mandatory adjunct ใน NF ทุกราย

**Second-look ≥ 48 hr** — ช้าเกิน  
→ NF progress ต่อเนื่อง → second-look 24 hr หลัง OR เสมอ

</Callout>
