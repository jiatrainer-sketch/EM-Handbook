## 🎯 Purpose

ประเมิน **level of consciousness** ใน trauma, stroke, metabolic encephalopathy, post-arrest, sepsis

**Range:** 3 (deep coma) → 15 (fully alert)

---

## 📊 Scoring

### Eye Opening (E) — 4 points

| Score | Response |
|---|---|
| **4** | Spontaneous |
| **3** | To voice (verbal command) |
| **2** | To pain |
| **1** | None |

### Verbal Response (V) — 5 points

| Score | Response |
|---|---|
| **5** | Oriented (person, place, time) |
| **4** | Confused conversation |
| **3** | Inappropriate words |
| **2** | Incomprehensible sounds |
| **1** | None |

### Motor Response (M) — 6 points

| Score | Response |
|---|---|
| **6** | Obeys commands |
| **5** | Localizes to pain |
| **4** | Withdraws from pain |
| **3** | Abnormal **flexion** (decorticate) |
| **2** | Abnormal **extension** (decerebrate) |
| **1** | None |

**Document: E_V_M_ = total** (e.g., E3V4M5 = 12)

---

## 🚦 Severity

| Total | Classification | Action |
|---|---|---|
| **13–15** | Mild | Observation, CT if indicated |
| **9–12** | Moderate | Admit, serial neuro, CT |
| **≤ 8** | **Severe** | **Intubate** (airway at risk), ICU, CT stat |

---

## ⚠️ Pitfalls

```
🚩 GCS ≤ 8 → intubate
   - Inability to protect airway
   - Usually need definitive airway

🚩 Intubated / aphasic patient
   - Record V as "T" (tube) or "A" (aphasic)
   - Max score = E4VTM6 = 10T
   - Use motor + eye alone for trend

🚩 Sedated patient
   - Must reassess when sedation off
   - Record "under sedation" in chart

🚩 Facial trauma / eyes swollen shut
   - Record E as "C" (closed)
   - Can't scored properly
   
🚩 Trend > absolute number
   - Decline by ≥ 2 points = significant
   - Call surgeon / neurosurgeon

🚩 Hypoglycemia / hypoxia mimics
   - Always check BS + SpO₂ first
   - Correct before labeling "severe TBI"

🚩 Toxic / metabolic causes
   - Alcohol, drug OD, uremia
   - Electrolyte (Na, Ca, Mg)
   - Ammonia (hepatic)
   - Endocrine (hypo, myxedema)
```

---

## 🔗 Quick use

**Document in chart:**
```
GCS 10 (E2V3M5)
Trend: 15 → 13 → 10 over 6 hr — WORSENING
```

**Red flags:**
- Drop ≥ 2 points
- GCS ≤ 8
- Anisocoria (unequal pupils)
- Cushing's triad (HTN + bradycardia + irregular resp)
- Decerebrate / decorticate

→ STAT CT brain + neurosurgery consult
